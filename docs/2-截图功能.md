## 1. 功能需求

### 1.1 核心功能列表

| 功能 ID | 功能名称 | 优先级 | 说明 |
|---------|----------|--------|------|
| SCR-001 | 全屏截图 | P0 | 捕获当前活动显示器全部内容 |
| SCR-002 | 区域截图 | P0 | 用户拖拽选择矩形区域 |
| SCR-003 | 窗口截图 | P1 | 自动识别并捕获指定窗口（含阴影裁剪） |
| SCR-004 | 延时截图 | P1 | 3 / 5 / 10 秒倒计时后自动截图 |
| SCR-005 | 滚动截图 | P2 | 捕获超出屏幕范围的可滚动内容（预留接口） |
| SCR-006 | 截图标注 | P0 | 矩形、椭圆、箭头、文字、马赛克、自由笔 |
| SCR-007 | 撤销/重做 | P0 | 对标注操作进行无限次撤销与重做 |
| SCR-008 | 保存到文件 | P0 | PNG / JPEG / WebP，支持自定义路径 |
| SCR-009 | 复制到剪贴板 | P0 | 直接将截图结果写入系统剪贴板 |
| SCR-010 | 截图历史 | P1 | 本次会话内的截图记录，可二次编辑 |
| SCR-011 | 多显示器支持 | P1 | 枚举所有显示器，支持跨屏区域选择 |
| SCR-012 | 高 DPI 支持 | P0 | 正确处理 HiDPI / Retina 屏幕缩放 |
| SCR-013 | 快捷键触发 | P0 | 全局热键（可自定义）触发各截图模式 |
| SCR-014 | 贴图（Pin）| P2 | 将截图结果置顶浮动显示在屏幕上 |

### 1.2 非功能需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 截图触发到覆盖层显示 | < 80ms | 从热键按下到选区 UI 出现 |
| 区域确认到图像可操作 | < 50ms | 从鼠标释放到标注工具可用 |
| 标注渲染帧率 | ≥ 60fps | 绘制过程中无卡顿 |
| 保存 PNG（1080p） | < 200ms | 含文件 I/O |
| 内存占用（空闲） | < 5MB | 未触发截图时模块内存基线 |
| 最大单张图片内存 | < 50MB | 4K 截图 + 标注层上限 |

### 1.3 用户操作流程

```
【区域截图标准流程】

用户按下热键 (Ctrl+Shift+A)
        │
        ▼
主程序隐藏/最小化（可选）
        │
        ▼
ScreenCapture 捕获全屏快照（作为覆盖层背景）
        │
        ▼
ScreenshotOverlay.qml 全屏展示
  ├── 鼠标十字准星
  ├── 放大镜（像素级精准定位）
  └── 当前坐标 / 色值提示
        │

        │
        ▼
SelectionRect 动态更新（实时尺寸提示）
        │
用户松开鼠标
        │
        ▼
AnnotationToolbar.qml 出现（工具栏）
        │
用户选择标注工具并操作（可多次）
        │
        ▼
用户点击【完成】
  ├──→ 复制到剪贴板
  ├──→ 保存到文件
  └──→ 贴图（Pin）
        │
        ▼
覆盖层关闭，截图存入历史
```

## 4. 数据结构

### 4.1 完整数据类型定义

```cpp
// src/services/screenshot/ScreenshotTypes.h

#pragma once
#include <QImage>
#include <QRect>
#include <QRectF>
#include <QPointF>
#include <QColor>
#include <QFont>
#include <QDateTime>
#include <QString>
#include <QScreen>
#include <vector>
#include <memory>
#include <variant>
#include <cstdint>

// ── ID 类型别名 ────────────────────────────────────────────────────────────
using ScreenshotId    = uint64_t;
using AnnotationId    = uint32_t;
using CommandId       = uint32_t;

// ── 枚举：截图模式 ────────────────────────────────────────────────────────
enum class CaptureMode : uint8_t {
    FullScreen  = 0,   // 捕获整个显示器
    Region      = 1,   // 用户框选区域
    Window      = 2,   // 指定窗口
    Delayed     = 3,   // 倒计时后自动截图（与上面三种组合使用）
};

// ── 枚举：导出格式 ────────────────────────────────────────────────────────
enum class ExportFormat : uint8_t {
    PNG  = 0,
    JPEG = 1,
    WebP = 2,
    BMP  = 3,
};

// ── 枚举：截图完成后的动作 ─────────────────────────────────────────────────
enum class FinalizeAction : uint8_t {
    SaveToFile        = 0,
    CopyToClipboard   = 1,
    SaveAndCopy       = 2,
    Pin               = 3,   // 贴图
    Discard           = 4,   // 取消
};

// ── 枚举：标注工具类型 ────────────────────────────────────────────────────
enum class AnnotationTool : uint8_t {
    Select    = 0,   // 选择/移动已有标注
    Rectangle = 1,
    Ellipse   = 2,
    Arrow     = 3,
    Line      = 4,
    Text      = 5,
    Mosaic    = 6,   // 马赛克
    Freehand  = 7,   // 自由笔
    Eraser    = 8,
    ColorPick = 9,   // 取色器
};

// ── 枚举：截图服务状态机 ──────────────────────────────────────────────────
enum class ScreenshotState : uint8_t {
    Idle        = 0,
    Pending     = 1,   // 等待用户选区（或等待延时）
    CountingDown= 2,   // 延时倒计时中
    Captured    = 3,   // 已捕获，可进入标注
    Annotating  = 4,   // 标注中
    Saving      = 5,   // 保存/导出中
};

// ── 截图请求 ──────────────────────────────────────────────────────────────
struct CaptureRequest {
    CaptureMode   mode         = CaptureMode::Region;
    int           screenIndex  = 0;         // 多显示器索引，-1 表示所有
    QRect         region;                   // Region 模式下生效，逻辑坐标
    WId           windowId     = 0;         // Window 模式下生效
    int           delayMs      = 0;         // Delayed 模式倒计时毫秒数
    bool          showCursor   = false;     // 是否捕获鼠标光标
    bool          captureDropShadow = false;// Window 模式是否包含阴影
};

// ── 显示器信息 ────────────────────────────────────────────────────────────
struct ScreenInfo {
    int     index;
    QString name;
    QRect   geometry;        // 逻辑坐标（Qt 坐标系）
    QRect   physicalGeometry;// 物理像素坐标
    double  dpr;             // Device Pixel Ratio（HiDPI 缩放比）
    bool    isPrimary;
};

// ═══════════════════════════════════════════════════════════════════════════
// 标注数据结构
// ═══════════════════════════════════════════════════════════════════════════

// ── 标注样式 ──────────────────────────────────────────────────────────────
struct AnnotationStyle {
    QColor  strokeColor  = Qt::red;
    QColor  fillColor    = Qt::transparent;
    float   strokeWidth  = 2.0f;
    Qt::PenStyle penStyle = Qt::SolidLine;
    QFont   font;
    bool    filled       = false;
    int     mosaicBlock  = 10;    // 马赛克块大小（像素）
    float   arrowHeadSize = 12.0f;
};

// ── 基础标注数据（所有形状公共字段）─────────────────────────────────────
struct AnnotationBase {
    AnnotationId   id{0};
    AnnotationTool tool{AnnotationTool::Rectangle};
    AnnotationStyle style;
    bool           selected{false};
    bool           locked{false};    // 锁定后不可移动/修改
    QDateTime      createdAt;
};

// ── 各形状数据（用 std::variant 统一存储）────────────────────────────────
struct RectShape {
    AnnotationBase base;
    QRectF         rect;            // 场景坐标（相对截图图像左上角）
};

struct EllipseShape {
    AnnotationBase base;
    QRectF         rect;
};

struct ArrowShape {
    AnnotationBase base;
    QPointF        start;
    QPointF        end;
};

struct LineShape {
    AnnotationBase base;
    QPointF        start;
    QPointF        end;
};

struct TextShape {
    AnnotationBase base;
    QPointF        position;        // 文字左上角
    QString        text;
    QRectF         boundingRect;    // 渲染后的边界（用于选中检测）
};

struct MosaicShape {
    AnnotationBase base;
    QRectF         rect;
    // 实际马赛克效果在 ShapeRenderer 中实时计算，不存储像素
};

struct FreehandShape {
    AnnotationBase          base;
    std::vector<QPointF>    points;  // 采样点序列
    QRectF                  boundingRect;
};

// 所有形状的 variant 类型
using AnnotationShape = std::variant<
    RectShape,
    EllipseShape,
    ArrowShape,
    LineShape,
    TextShape,
    MosaicShape,
    FreehandShape
>;

// ── 标注层（一个 Screenshot 持有一个 AnnotationLayer）──────────────────
struct AnnotationLayer {
    std::vector<AnnotationShape> shapes;
    // z-order = shapes 的索引顺序（后面的在上层）

    [[nodiscard]] std::optional<std::reference_wrapper<AnnotationShape>>
    findById(AnnotationId id);

    [[nodiscard]] AnnotationId nextId() const noexcept;
};

// ═══════════════════════════════════════════════════════════════════════════
// 截图主数据结构
// ═══════════════════════════════════════════════════════════════════════════
struct Screenshot {
    ScreenshotId    id{0};
    QImage          rawImage;         // 原始截图（未加标注），物理像素
    QRect           capturedRegion;   // 在逻辑坐标系中的位置
    double          dpr{1.0};         // 来源显示器的 DPR
    QDateTime       capturedAt;
    CaptureMode     mode{CaptureMode::FullScreen};
    AnnotationLayer annotations;
    QString         savedPath;        // 已保存时非空
    bool            copiedToClipboard{false};

    // ── 导出方法 ───────────────────────────────────────────────────────
    /**
     * @brief 将 rawImage 与所有标注合并，返回最终图像
     * @param targetSize 为空则使用原始尺寸（物理像素）
     */
    [[nodiscard]] QImage flatten(const QSize& targetSize = {}) const;

    /**
     * @brief 返回截图的逻辑尺寸（供 QML 显示用）
     */
    [[nodiscard]] QSize logicalSize() const noexcept {
        return (rawImage.size() / dpr).toSize();
    }
};
```

## 5. 平台原生截图层（ScreenCapture）

### 5.1 抽象接口

```cpp
// src/services/screenshot/capture/IScreenCapture.h

#pragma once
#include <QImage>
#include <QRect>
#include <QScreen>
#include <functional>
#include <vector>
#include "core/error/Error.h"
#include "../ScreenshotTypes.h"

/**
 * @brief 平台原生屏幕捕获抽象接口
 *
 * 各平台实现此接口，屏蔽平台差异。
 * 所有方法均在调用方线程执行（截图线程，非 GUI 线程）。
 */
class IScreenCapture {
public:
    virtual ~IScreenCapture() = default;

    /**
     * @brief 枚举当前所有显示器
     */
    [[nodiscard]] virtual std::vector<ScreenInfo>
    enumerateScreens() const = 0;

    /**
     * @brief 捕获指定显示器全部内容（同步）
     *
     * @param screenIndex  显示器索引（来自 enumerateScreens）
     * @param showCursor   是否包含鼠标指针
     * @return 物理像素 QImage（Format_ARGB32_Premultiplied）
     */
    [[nodiscard]] virtual Result<QImage>
    captureScreen(int screenIndex, bool showCursor = false) = 0;

    /**
     * @brief 捕获指定逻辑矩形区域
     *
     * @param region  逻辑坐标矩形（Qt 全局坐标系）
     */
    [[nodiscard]] virtual Result<QImage>
    captureRegion(const QRect& region, bool showCursor = false) = 0;

    /**
     * @brief 捕获指定窗口
     *
     * @param windowId  平台窗口句柄（WId）
     * @param includeDropShadow  是否包含系统阴影
     */
    [[nodiscard]] virtual Result<QImage>
    captureWindow(WId windowId, bool includeDropShadow = false) = 0;

    /**
     * @brief 枚举所有顶层窗口（供窗口截图模式使用）
     */
    struct WindowInfo {
        WId     id;
        QString title;
        QRect   geometry;   // 逻辑坐标
        QPixmap thumbnail;  // 小缩略图（可选，空则不显示）
    };

    [[nodiscard]] virtual std::vector<WindowInfo>
    enumerateWindows() const = 0;

    /**
     * @brief 获取鼠标当前所在窗口（用于窗口高亮）
     */
    [[nodiscard]] virtual std::optional<WindowInfo>
```

```cpp
// src/services/screenshot/capture/win/Win32ScreenCapture.cpp （关键实现节选）

#ifdef Q_OS_WIN
#include "Win32ScreenCapture.h"
#include "core/log/Logger.h"
#include "core/log/LogModules.h"
#include <QGuiApplication>
#include <dwmapi.h>
#pragma comment(lib, "dwmapi.lib")
#pragma comment(lib, "d3d11.lib")
#pragma comment(lib, "dxgi.lib")

Result<QImage> Win32ScreenCapture::captureScreen(int screenIndex, bool showCursor) {
    TB_LOG_DEBUG(LogModule::Screenshot,
        "captureScreen: index={}, cursor={}", screenIndex, showCursor);

    auto screens = enumerateScreens();
    if (screenIndex < 0 || screenIndex >= static_cast<int>(screens.size())) {
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::InvalidArgument,
            "Invalid screen index: " + std::to_string(screenIndex)));
    }

    // 优先 DXGI
    if (m_dxgiAvailable && screenIndex < static_cast<int>(m_outputs.size())) {
        auto result = captureViaDxgi(m_outputs[screenIndex]);
        if (result.isOk()) {
            if (showCursor) drawCursorOnImage(result.value());
            return result;
        }
        TB_LOG_WARN(LogModule::Screenshot,
            "DXGI capture failed, falling back to BitBlt: {}",
            result.error().message);
    }

    // BitBlt 回退
    const auto& screen = screens[screenIndex];
    return captureViaBitBlt(nullptr, screen.physicalGeometry);
}

Result<QImage> Win32ScreenCapture::captureWindow(WId windowId, bool includeDropShadow) {
    HWND hwnd = reinterpret_cast<HWND>(windowId);
    if (!IsWindow(hwnd)) {
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::InvalidArgument, "Invalid window handle"));
    }

    RECT winRect{};
    if (includeDropShadow) {
        // DwmGetWindowAttribute 获取含阴影的扩展边界
        DwmGetWindowAttribute(hwnd, DWMWA_EXTENDED_FRAME_BOUNDS,
                              &winRect, sizeof(RECT));
    } else {
        GetWindowRect(hwnd, &winRect);
    }

    QRect region(winRect.left, winRect.top,
                 winRect.right  - winRect.left,
                 winRect.bottom - winRect.top);

    // PrintWindow 方式（支持最小化窗口）
    int width  = region.width();
    int height = region.height();

    HDC hdc      = GetDC(nullptr);
    HDC memDC    = CreateCompatibleDC(hdc);
    HBITMAP hBmp = CreateCompatibleBitmap(hdc, width, height);
    auto oldBmp  = SelectObject(memDC, hBmp);

    // PW_RENDERFULLCONTENT: 捕获 GPU 渲染内容（Win8.1+）
    BOOL ok = PrintWindow(hwnd, memDC, PW_RENDERFULLCONTENT);

    QImage image;
    if (ok) {
        image = QImage(width, height, QImage::Format_ARGB32);
        BITMAPINFOHEADER bi{};
        bi.biSize        = sizeof(BITMAPINFOHEADER);
        bi.biWidth       = width;
        bi.biHeight      = -height;   // 负值 = 顶部为起点
        bi.biPlanes      = 1;
        bi.biBitCount    = 32;
        bi.biCompression = BI_RGB;
        GetDIBits(hdc, hBmp, 0, height,
                  image.bits(), reinterpret_cast<BITMAPINFO*>(&bi),
                  DIB_RGB_COLORS);
    }

    SelectObject(memDC, oldBmp);
    DeleteObject(hBmp);
    DeleteDC(memDC);
    ReleaseDC(nullptr, hdc);

    if (!ok) {
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::ScreenCaptureError, "PrintWindow failed"));
    }
    return Result<QImage>::ok(std::move(image));
}

#endif // Q_OS_WIN
```

```objc
    CGImageRelease(cgImage);

    if (result.isNull()) {
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::ScreenCaptureError, "Failed to convert CGImage to QImage"));
    }
    return Result<QImage>::ok(std::move(result));

// CGImage → QImage 转换（避免逐像素拷贝，使用 CGBitmapContext）
QImage MacScreenCapture::cgImageToQImage(CGImageRef cgImage) {
    const size_t width  = CGImageGetWidth(cgImage);
    const size_t height = CGImageGetHeight(cgImage);

    QImage image(static_cast<int>(width),
                 static_cast<int>(height),
                 QImage::Format_ARGB32_Premultiplied);
    image.fill(Qt::transparent);

    CGColorSpaceRef colorSpace = CGColorSpaceCreateDeviceRGB();
    CGContextRef ctx = CGBitmapContextCreate(
        image.bits(), width, height, 8,
        image.bytesPerLine(), colorSpace,
        kCGImageAlphaPremultipliedFirst | kCGBitmapByteOrder32Host);

    CGContextDrawImage(ctx, CGRectMake(0, 0, width, height), cgImage);
    CGContextRelease(ctx);
    CGColorSpaceRelease(colorSpace);
    return image;
}
```

### 5.4 Linux 实现（X11 + XShm）

```cpp
// src/services/screenshot/capture/x11/X11ScreenCapture.cpp（关键节选）

#ifdef Q_OS_LINUX
#include "X11ScreenCapture.h"
#include <X11/Xlib.h>
#include <X11/extensions/XShm.h>
#include <sys/shm.h>

Result<QImage> X11ScreenCapture::captureScreen(int screenIndex, bool showCursor) {
    Display* display = XOpenDisplay(nullptr);
    if (!display) {
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::ScreenCaptureError, "Cannot open X display"));
    }

    auto screens = enumerateScreens();
    if (screenIndex >= static_cast<int>(screens.size())) {
        XCloseDisplay(display);
        return Result<QImage>::err(TB_MAKE_ERROR(
            ErrorCode::InvalidArgument, "Invalid screen index"));
    }

    const auto& si = screens[screenIndex];
    Window root    = RootWindow(display, screenIndex);
    int    width   = si.physicalGeometry.width();
    int    height  = si.physicalGeometry.height();

    QImage image;

    // 优先使用 XShm（共享内存，零拷贝）
    int shmMajor, shmMinor;
    Bool shmPixmaps;
    if (XShmQueryVersion(display, &shmMajor, &shmMinor, &shmPixmaps)) {
        XShmSegmentInfo shmInfo{};
        XImage* ximg = XShmCreateImage(display,
            DefaultVisual(display, screenIndex),
            DefaultDepth(display, screenIndex),
            ZPixmap, nullptr, &shmInfo, width, height);

        shmInfo.shmid = shmget(IPC_PRIVATE,
            static_cast<size_t>(ximg->bytes_per_line * ximg->height),
            IPC_CREAT | 0600);
        shmInfo.shmaddr = ximg->data =
            static_cast<char*>(shmat(shmInfo.shmid, nullptr, 0));
        shmInfo.readOnly = False;

        XShmAttach(display, &shmInfo);
        XShmGetImage(display, root, ximg, 0, 0, AllPlanes);
        XSync(display, False);

        // 直接从共享内存构建 QImage（无额外拷贝）
        image = QImage(
            reinterpret_cast<const uchar*>(shmInfo.shmaddr),
            width, height, ximg->bytes_per_line,
            QImage::Format_RGB32).copy();  // copy() 脱离共享内存生命周期

        XShmDetach(display, &shmInfo);
        XDestroyImage(ximg);
        shmdt(shmInfo.shmaddr);
        shmctl(shmInfo.shmid, IPC_RMID, nullptr);
    } else {
        // 回退：XGetImage
        XImage* ximg = XGetImage(display, root, 0, 0, width, height,
                                  AllPlanes, ZPixmap);
        if (!ximg) {
            XCloseDisplay(display);
            return Result<QImage>::err(TB_MAKE_ERROR(
                ErrorCode::ScreenCaptureError, "XGetImage failed"));
        }
        image = QImage(reinterpret_cast<const uchar*>(ximg->data),
                       width, height, ximg->bytes_per_line,
                       QImage::Format_RGB32).copy();
        XDestroyImage(ximg);
    }

    XCloseDisplay(display);
    return Result<QImage>::ok(std::move(image));
}
#endif // Q_OS_LINUX
```

### 9.2 文件名模板解析

```cpp
// ScreenshotStorage.cpp
QString ScreenshotStorageImpl::generateFileName(
    const QString& templ, ExportFormat format) const
{
    // 支持的模板变量：
    // {yyyy}  年（4位）     {MM}   月（2位）  {dd}  日（2位）
    // {HH}    时（24小时）  {mm}   分（2位）  {ss}  秒（2位）
    // {ms}    毫秒（3位）   {idx}  自增序号   {screen} 显示器名

    QDateTime now = QDateTime::currentDateTime();
    QString name  = templ;

    name.replace(QStringLiteral("{yyyy}"), now.toString(QStringLiteral("yyyy")));
    name.replace(QStringLiteral("{MM}"),   now.toString(QStringLiteral("MM")));
    name.replace(QStringLiteral("{dd}"),   now.toString(QStringLiteral("dd")));
    name.replace(QStringLiteral("{HH}"),   now.toString(QStringLiteral("HH")));
    name.replace(QStringLiteral("{mm}"),   now.toString(QStringLiteral("mm")));
    name.replace(QStringLiteral("{ss}"),   now.toString(QStringLiteral("ss")));
    name.replace(QStringLiteral("{ms}"),   now.toString(QStringLiteral("zzz")));
    name.replace(QStringLiteral("{idx}"),  QString::number(++m_fileIndex));

    // 附加扩展名
    static const QMap<ExportFormat, QString> extMap {
        { ExportFormat::PNG,  QStringLiteral(".png")  },
        { ExportFormat::JPEG, QStringLiteral(".jpg")  },
        { ExportFormat::WebP, QStringLiteral(".webp") },
        { ExportFormat::BMP,  QStringLiteral(".bmp")  },
    };
    return name + extMap.value(format, QStringLiteral(".png"));
}
```

```qml
import QtQuick.Controls 2.15
import QtQuick.Window 2.15
import ToolBox.ViewModels 1.0

/*!
    \qmltype ScreenshotOverlay
    \brief 全屏截图覆盖层
*/
Window {
    id: root

    // ── 窗口属性 ──────────────────────────────────────────────────────────
    flags: Qt.FramelessWindowHint
           | Qt.WindowStaysOnTopHint
           | Qt.Tool
           | Qt.BypassWindowManagerHint  // Linux: 跳过窗口管理器
    visibility: Window.FullScreen
    color: "transparent"
    visible: ScreenshotViewModel.isOverlayVisible

    // ── 键盘焦点 ──────────────────────────────────────────────────────────
    onVisibleChanged: if (visible) requestActivate()

    Keys.onEscapePressed:          ScreenshotViewModel.cancel()
    Keys.onReturnPressed:          ScreenshotViewModel.finalizeSaveAndCopy()
    Keys.onPressed: (event) => handleGlobalKey(event)

    function handleGlobalKey(event) {
        const ctrl = event.modifiers & Qt.ControlModifier
        if (ctrl && event.key === Qt.Key_Z) {
            event.modifiers & Qt.ShiftModifier
                ? ScreenshotViewModel.redo()
                : ScreenshotViewModel.undo()
            event.accepted = true
        } else if (ctrl && event.key === Qt.Key_Y) {
            ScreenshotViewModel.redo(); event.accepted = true
        }
    }

    // ── 背景：全屏快照 + 半透明遮罩 ──────────────────────────────────────
    Image {
        id: bgImage
        anchors.fill: parent
        source:       "image://screenshot/background"   // Qt image provider
        cache:        false

        // 半透明遮罩（选区外区域变暗）
        Rectangle {
            anchors.fill: parent
            color: "#80000000"  // 50% 黑色
            visible: selectionLayer.hasSelection
                     && !annotationToolbar.visible

            // 注意：QML 暂无 clip-path，用 4 块遮罩矩形实现
        }

                     ? Qt.ArrowCursor
                     : Qt.CrossCursor

        onPressed: (mouse) => {
            startPos = Qt.point(mouse.x, mouse.y)
            selectionLayer.startSelection(startPos)
        }
        onPositionChanged: (mouse) => {
            if (pressed)
                selectionLayer.updateSelection(Qt.point(mouse.x, mouse.y))
            magnifier.updatePosition(Qt.point(mouse.x, mouse.y))
        }
        onReleased: (mouse) => {
            selectionLayer.commitSelection(Qt.point(mouse.x, mouse.y))
        }
    }

    // ── 选区层 ────────────────────────────────────────────────────────────

        id: selectionLayer
        anchors.fill: parent

        onSelectionCommitted: (rect) => {
            ScreenshotViewModel.setSelectionRect(rect)
        }
    }

                 && appSettings.showMagnifier

        onMouseReleased:  (x, y) => ScreenshotViewModel.onAnnotationMouseRelease(x, y)
```

## 12. 线程模型与时序

### 12.1 线程分工

```
GUI Thread（Qt 主线程）
  职责：QML 渲染、ViewModel 方法调用、信号处理
  禁止：任何超过 5ms 的同步操作

Screenshot Worker Thread（专用截图线程）
  职责：IScreenCapture 所有平台 API 调用（可能阻塞）
  创建：Application::init() 时创建并启动
  通信：通过 Qt QueuedConnection 信号返回结果给主线程

Annotation Thread（可选，复杂标注计算）
  职责：大尺寸马赛克预计算、导出渲染合并
  创建：按需，使用 AppContext ThreadPool

File I/O Thread（共享 ThreadPool）
  职责：截图文件保存（Qt::QueuedConnection 回调结果）
```

### 17.1 关键路径性能要求

| 操作 | 目标耗时 | 测量方式 |
|------|----------|----------|

### 17.2 内存使用约束

```
单张 1080p 截图（rawImage）：
  1920 × 1080 × 4 bytes = 8.3 MB（ARGB32）

单张 4K 截图：
  3840 × 2160 × 4 bytes = 33.2 MB

标注层（20 个标注，FreehandShape 除外）：
  < 50KB（仅坐标数据）

历史缩略图（200×150，20张）：
  200 × 150 × 4 × 20 ≈ 2.4 MB

  rawImage + flatten() 临时图 + 导出缓冲区 ≈ 3 × 8.3MB ≈ 25MB（1080p）
```

```cpp
// 优化1：flatten() 使用 QPainter 而非逐像素操作
QImage Screenshot::flatten(const QSize& targetSize) const {
    QSize sz = targetSize.isValid() ? targetSize : rawImage.size();
    QImage result(sz, QImage::Format_ARGB32_Premultiplied);

    QPainter p(&result);
    p.setRenderHint(QPainter::SmoothPixmapTransform);
    p.setRenderHint(QPainter::Antialiasing);

    // 绘制原图
    p.drawImage(result.rect(), rawImage, rawImage.rect());

    // 通过 ShapeRenderer 绘制标注层（无中间 QImage，直接绘制到 result）
    ShapeRenderer renderer;
    for (const auto& shape : annotations.shapes)
        renderer.render(p, shape, rawImage, 1.0f);

    p.end();
    return result;
}
```

```cpp
bool X11ScreenCapture::isWaylandSession() {

        || qgetenv("WAYLAND_DISPLAY") != "";

Result<QImage> X11ScreenCapture::captureScreen(int screenIndex, bool showCursor) {
    if (isWaylandSession()) {
        // 当前版本：通过 Qt::AA_UseSoftwareOpenGL + QScreen::grabWindow 回退
        // 局限：某些合成器下仅能捕获本进程窗口
        TB_LOG_WARN(LogModule::Screenshot,
            "Wayland session detected: using QScreen::grabWindow fallback. "
            "Full Wayland support (wlr-screencopy) planned for v1.2");

        QScreen* screen = QGuiApplication::screens().value(screenIndex);
        if (!screen)
            return Result<QImage>::err(TB_MAKE_ERROR(
                ErrorCode::ScreenCaptureInvalidDisplay, "Invalid screen"));

        if (img.isNull())

                ErrorCode::ScreenCaptureError, "QScreen::grabWindow failed on Wayland"));
        return Result<QImage>::ok(std::move(img));
```
