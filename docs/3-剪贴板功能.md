## 1. 功能概述

### 1.1 核心职责

剪贴板模块负责：

- **系统剪贴板监听**：实时捕获系统剪贴板变化（文本、图片、文件路径）
- **历史管理**：持久化存储剪贴板历史，支持快速恢复
- **内容分类**：按类型（文本、图片、文件）组织条目
- **高效查询**：全文搜索、标签过滤、时间范围查询
- **用户交互**：置顶常用项、删除不需要项、批量操作
- **数据同步**（预留）：多设备剪贴板同步

### 1.2 性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 监听响应延迟 | < 100ms | 剪贴板变化到界面更新 |
| 单次插入耗时 | < 50ms | SQLite 持久化 |
| 查询响应时间 | < 200ms | 搜索 500 条历史记录 |
| 内存占用 | < 100MB | 保存 500 条记录（含图片缩略图） |
| 启动恢复时间 | < 1s | 从磁盘加载历史数据 |

### 1.3 支持的内容类型

```
┌─ 文本 (Text)
│  ├─ 纯文本
│  └─ HTML 富文本
├─ 图片 (Image)
│  ├─ 以 PNG 格式压缩存储
│  └─ 保留宽度、高度、缩略图
├─ 文件路径 (Files)
│  └─ 多文件路径列表
└─ 自定义类型 (Custom)
   └─ 预留扩展槽位
```

## 2. 需求分析

### 2.1 功能需求

#### FR1：基础监听与存储
- 实时监听系统剪贴板变化
- 自动保存新条目到 SQLite 数据库
- 启动时恢复历史记录
- 支持最多 500 条记录（可配置）

#### FR2：界面展示
- 列表展示所有历史记录
- 显示条目类型、内容预览、时间戳
- 大图片以缩略图展示

#### FR3：用户交互
- 单击条目将其恢复到系统剪贴板
- 右键菜单：复制、删除、置顶/取消置顶、标签编辑
- 搜索框：全文搜索条目
- 批量删除选中项

#### FR4：分类与管理
- 按类型过滤（全部、文本、图片、文件）
- 标签管理：为条目添加自定义标签
- 置顶：常用项置顶显示
- 分类快捷键

#### FR5：配置与持久化
- 配置最大历史数量
- 配置是否自动监听图片
- 图片大小限制（防止 OOM）

### 2.2 非功能需求

| 类别 | 需求 |
|------|------|
| 性能 | 监听响应 < 100ms，查询响应 < 200ms |
| 可用性 | 支持键盘快捷键操作 |
| 安全 | 不记录密码输入（通过黑名单应用） |
| 可扩展性 | 支持插件扩展（标签规则、内容过滤） |
| 跨平台 | Windows / macOS / Linux |

## 3. 架构设计

### 3.1 分层架构

```
┌─────────────────────────────────────────────────────────────┐
│                   QML UI Layer                              │
│      ClipboardPage.qml / ClipboardListView.qml             │
└────────────────────┬────────────────────────────────────────┘
                     │ (Q_PROPERTY / Q_INVOKABLE / signals)
┌────────────────────▼────────────────────────────────────────┐
│              ViewModel Layer (QObject)                       │
│            ClipboardViewModel.h/.cpp                        │
│  ┌──────────────────┬──────────────────────────────────┐    │
│  │ List Model       │ Item Model (QAbstractListModel) │    │
│  │ Search/Filter    │ ClipboardListModel              │    │
│  │ User Actions     │                                 │    │
│  └──────────────────┴──────────────────────────────────┘    │
└────────────────────┬────────────────────────────────────────┘
                     │ (C++ 函数调用 / std::function 回调)
┌────────────────────▼────────────────────────────────────────┐
│           Service Layer (Pure C++)                          │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ ClipboardService     — 业务逻辑聚合                  │   │
│  │ ClipboardMonitor     — 系统剪贴板监听                │   │
│  │ ClipboardStorage     — SQLite 持久化                 │   │
│  │ ClipboardProcessor   — 内容处理（压缩、缩略图）      │   │
│  └──────────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────────────────┐
│        Core Infrastructure                                  │
│  Logger / EventBus / ConfigManager / ThreadPool            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 数据流

```
系统剪贴板变化
    ↓
[ClipboardMonitor] 监听线程
    ↓
内容类型识别 + 内容处理（压缩、缩略图）
    ↓
[ClipboardStorage] 数据库线程
    ↓
SQLite 持久化
    ↓
[EventBus::publish(ClipboardChangedEvent)]
    ↓
[ClipboardViewModel] 主线程
    ↓
[ClipboardListModel] 更新数据
    ↓
QML → 界面刷新
```

### 3.3 线程模型

```
主线程 (Qt GUI Thread):
  - ViewModel 的 slot/signal
  - QML UI 操作
  - 禁止阻塞 > 16ms

监听线程 (ClipboardMonitor):
  - 系统剪贴板监听（QClipboard）
  - 内容初步解析
  - 发布事件到 EventBus

处理线程 (WorkerThread):
  - 图片压缩、生成缩略图
  - SQLite 插入/查询

数据库线程 (ThreadPool):
  - SQLite 同步读写（数据库本身加锁）
```

### 4.2 模块职责

| 模块 | 职责 | 依赖 | 线程安全 |
|------|------|------|---------|
| `ClipboardService` | 业务逻辑聚合、生命周期管理 | Storage, Monitor, Processor | 是（内部协调） |

| `ClipboardListModel` | QML 列表数据模型 | ClipboardItem | 是（主线程）|

## 5. 数据结构定义

### 5.1 基础类型

```cpp
// src/services/clipboard/ClipboardTypes.h

#pragma once
#include <QString>
#include <QByteArray>
#include <QDateTime>
#include <QStringList>
#include <QSize>
#include <cstdint>

// ── 剪贴板条目 ID ────────────────────────────────────────────────────────
using ClipboardItemId = uint64_t;

// ── 内容类型枚举 ────────────────────────────────────────────────────────
enum class ClipboardContentType : uint8_t {
    Text    = 0,      // 纯文本 + HTML
    Image   = 1,      // 图片数据
    Files   = 2,      // 文件路径列表
    Custom  = 255     // 预留自定义类型
};

// ── 文本内容 ────────────────────────────────────────────────────────────
struct ClipboardTextContent {
    QString text;           // 完整内容（HTML 或纯文本）
    QString plainText;      // 纯文本备份（用于搜索/预览）
    bool    isHtml{false};  // 是否为 HTML
};

// ── 图片内容 ────────────────────────────────────────────────────────────
struct ClipboardImageContent {
    QByteArray  pngData;      // PNG 压缩数据
    QByteArray  thumbnailData; // JPEG 缩略图（200x200）
    int         width{0};
    int         height{0};
    QString     sourceFormat;  // 原始格式（"PNG", "JPEG", "BMP"）
};

// ── 文件路径内容 ────────────────────────────────────────────────────────
struct ClipboardFilesContent {
    QStringList paths;
};

// ── 内容联合体 ──────────────────────────────────────────────────────────
using ClipboardContent = std::variant<
    ClipboardTextContent,
    ClipboardImageContent,
    ClipboardFilesContent
>;

// ── 剪贴板条目结构 ──────────────────────────────────────────────────────
struct ClipboardItem {
    ClipboardItemId     id{0};
    ClipboardContentType type{ClipboardContentType::Text};
    ClipboardContent    content;
    
    // ── 元数据 ────────────────────────────────────────────────────────
    QString             preview;        // 用于 QML 展示的摘要（≤ 200 字符）
    QDateTime           createdAt;      // 创建时间
    QDateTime           accessedAt;     // 最后访问时间（用于 LRU 排序）
    bool                pinned{false};  // 是否置顶
    QStringList         tags;           // 用户自定义标签
    int                 accessCount{0}; // 被恢复次数
    
    // ── 工厂方法 ────────────────────────────────────────────────────────
    [[nodiscard]] static ClipboardItem fromText(
        const QString& text,
        bool isHtml = false);
    
    [[nodiscard]] static ClipboardItem fromImage(
        const QImage& image,
        const QString& sourceFormat = "PNG");
    
    [[nodiscard]] static ClipboardItem fromFiles(
        const QStringList& paths);
    
    // ── 工具方法 ────────────────────────────────────────────────────────
    [[nodiscard]] QString typeString() const noexcept;
    [[nodiscard]] bool    isEmpty() const noexcept;
    [[nodiscard]] size_t  estimatedSize() const noexcept; // 字节数
};

// ── 搜索过滤选项 ────────────────────────────────────────────────────────
struct ClipboardSearchOptions {
    QString           keyword;           // 搜索关键词
    QStringList       tags;              // 标签过滤（AND）
    ClipboardContentType typeFilter = static_cast<ClipboardContentType>(255); // 255 = 全部
    bool              pinnedOnly{false}; // 仅显示置顶项
    int               limitResults{50};  // 最多返回结果数
};

// ── 排序方式 ────────────────────────────────────────────────────────────
enum class ClipboardSortBy : uint8_t {
    CreatedAtDesc     = 0,  // 最新优先
    AccessedAtDesc    = 1,  // 最近使用优先
    AccessCountDesc   = 2,  // 使用频率最高优先
    PinnedFirst       = 3   // 置顶优先，然后按创建时间
};
```

## 6. 核心接口规范

### 6.1 ClipboardService 接口

```cpp
// src/services/clipboard/ClipboardService.h

#pragma once
#include <QString>
#include <QStringList>
#include <memory>
#include <functional>
#include "ClipboardItem.h"
#include "ClipboardTypes.h"
#include "core/error/Error.h"
#include "core/event/EventBus.h"

class ClipboardMonitor;
class ClipboardStorage;
class ClipboardProcessor;

/**
 * @brief 剪贴板服务门面类
 *
 * 负责协调监听、存储、处理三个子模块。
 * 对外暴露统一的业务接口。
 *
 * 线程安全：所有公开方法可从任意线程调用。
 */
class ClipboardService : public QObject {
    Q_OBJECT

public:
    using ClipboardCallback = std::function<void(Result<ClipboardItem>)>;
    using ItemListCallback  = std::function<void(Result<QList<ClipboardItem>>)>;

    explicit ClipboardService(QObject* parent = nullptr);
    ~ClipboardService() override;

    // ── 生命周期 ────────────────────────────────────────────────────────
    /**
     * @brief 初始化服务
     *
     * 加载 SQLite 数据库，启动监听线程，从配置加载历史记录。
     *
     * @return 初始化是否成功
     */
    [[nodiscard]] Result<void> initialize();

    /**
     * @brief 关闭服务
     *
     * 停止监听，关闭数据库，等待后台线程。
     */
    void shutdown() noexcept;

    // ── 查询接口 ────────────────────────────────────────────────────────
    /**
     * @brief 获取所有历史记录（最新优先）
     *
     * @param offset 分页偏移
     * @param limit  最多返回条数
     * @return 条目列表
     */
    [[nodiscard]] Result<QList<ClipboardItem>>
    getHistory(int offset = 0, int limit = 50) noexcept;

    /**
     * @brief 按 ID 获取单个条目
     *
     * @param id 条目 ID
     * @return 条目数据
     */
    [[nodiscard]] Result<ClipboardItem>
    getItem(ClipboardItemId id) noexcept;

    /**
     * @brief 搜索与过滤
     *
     * @param options 搜索选项（关键词、标签、类型等）
     * @return 过滤后的条目列表
     */
    [[nodiscard]] Result<QList<ClipboardItem>>
    search(const ClipboardSearchOptions& options) noexcept;

    /**
     * @brief 获取所有标签（用于 UI 下拉菜单）
     *
     * @return 标签列表
     */
    [[nodiscard]] Result<QStringList> getAllTags() noexcept;

    // ── 操作接口 ────────────────────────────────────────────────────────
    /**
     * @brief 将指定条目复制到系统剪贴板
     *
     * @param id 条目 ID
     * @return 操作是否成功
     */
    [[nodiscard]] Result<void>
    restoreToClipboard(ClipboardItemId id) noexcept;

    /**
     * @brief 删除指定条目
     *
     * @param id 条目 ID
     * @return 操作是否成功
     */
    [[nodiscard]] Result<void>
    deleteItem(ClipboardItemId id) noexcept;

    /**
     * @brief 批量删除
     *
     * @param ids 条目 ID 列表
     * @return 成功删除数
     */
    [[nodiscard]] Result<int>
    deleteItems(const QList<ClipboardItemId>& ids) noexcept;

    /**
     * @brief 清空所有历史记录
     *
     * @return 操作是否成功
     */
    [[nodiscard]] Result<void>
    clearAll() noexcept;

    /**
     * @brief 切换条目的置顶状态
     *
     * @param id     条目 ID
     * @param pinned 是否置顶
     * @return 操作是否成功
     */
    [[nodiscard]] Result<void>
    setPinned(ClipboardItemId id, bool pinned) noexcept;

    /**
     * @brief 为条目添加或修改标签
     *
     * @param id   条目 ID
     * @param tags 标签列表
     * @return 操作是否成功
     */
    [[nodiscard]] Result<void>
    setTags(ClipboardItemId id, const QStringList& tags) noexcept;

    // ── 监听控制 ────────────────────────────────────────────────────────
    /**
     * @brief 启动监听
     */
    void startMonitoring() noexcept;

    /**
     * @brief 停止监听
     */
    void stopMonitoring() noexcept;

    /**
     * @brief 检查是否正在监听
     */
    [[nodiscard]] bool isMonitoring() const noexcept;

    // ── 统计信息 ────────────────────────────────────────────────────────
    /**
     * @brief 获取统计信息
     *
     * @return 统计数据（总条数、占用空间等）
     */
    struct Statistics {
        int totalItems{0};
        int totalSizeKB{0};
        int textItems{0};
        int imageItems{0};
        int fileItems{0};
    };

    [[nodiscard]] Result<Statistics>
    getStatistics() noexcept;

private:
    void onClipboardChanged();
    void onMonitorError(const QString& error);
    void onStorageError(const QString& error);

    std::shared_ptr<ClipboardMonitor>   m_monitor;
    std::shared_ptr<ClipboardStorage>   m_storage;
    std::shared_ptr<ClipboardProcessor> m_processor;
    bool                                m_monitoring{false};
};
```

### 6.2 ClipboardMonitor 接口

```cpp
// src/services/clipboard/ClipboardMonitor.h

#pragma once
#include <QString>
#include <QObject>
#include <QClipboard>
#include <memory>
#include "ClipboardItem.h"
#include "core/error/Error.h"

/**
 * @brief 系统剪贴板监听类
 *
 * 通过 QClipboard 的 dataChanged 信号监听系统剪贴板变化。
 * 在独立线程中运行，避免阻塞主线程。
 *
 * 线程模型：可从任意线程调用 public 方法，内部使用 Qt 的线程亲和性。
 */
class ClipboardMonitor : public QObject {
    Q_OBJECT

public:
    explicit ClipboardMonitor(QObject* parent = nullptr);
    ~ClipboardMonitor() override;

    /**
     * @brief 启动监听
     *
     * @return 启动是否成功
     */
    [[nodiscard]] Result<void> start() noexcept;

    /**
     * @brief 停止监听
     */
    void stop() noexcept;

    /**
     * @brief 检查监听状态
     */
    [[nodiscard]] bool isRunning() const noexcept;

    /**
     * @brief 手动获取当前系统剪贴板内容
     *
     * @return 解析后的条目
     */
    [[nodiscard]] Result<ClipboardItem>
    getCurrentContent() const noexcept;

signals:
    /**
     * @brief 检测到剪贴板变化
     *
     * @param item 新条目数据
     */
    void contentChanged(const ClipboardItem& item);
```

## 7. 实现细节

### 7.1 系统剪贴板监听（跨平台）

```cpp
// src/services/clipboard/ClipboardMonitor.cpp

#include "ClipboardMonitor.h"
#include <QApplication>
#include <QClipboard>
#include <QMimeData>
#include <QImage>
#include <QUrl>
#include "core/log/Logger.h"
#include "core/log/LogModules.h"

Result<ClipboardItem> ClipboardMonitor::getCurrentContent() const noexcept {
    try {
        QClipboard* clipboard = QApplication::clipboard();
        if (!clipboard) {
            return Result<ClipboardItem>::err(
                TB_MAKE_ERROR(ErrorCode::Unknown, "QClipboard not available"));
        }

        const QMimeData* mimeData = clipboard->mimeData();
        if (!mimeData) {
            return Result<ClipboardItem>::err(
                TB_MAKE_ERROR(ErrorCode::ClipboardEmpty, "Clipboard is empty"));
        }

        // ── 优先级顺序：Image → Files → Html → Text ────────────────────
        
        // 优先检查图片
        if (mimeData->hasImage()) {
            QImage image = qvariant_cast<QImage>(mimeData->imageData());
            if (!image.isNull()) {
                TB_LOG_DEBUG(LogModule::Clipboard,
                    "Detected image clipboard: {}x{}", image.width(), image.height());
                return Result<ClipboardItem>::ok(
                    ClipboardItem::fromImage(image));
            }
        }

        // 检查文件路径
        if (mimeData->hasUrls()) {
            QStringList paths;
            for (const auto& url : mimeData->urls()) {
                if (url.isLocalFile()) {
                    paths.append(url.toLocalFile());
                }
            }
            if (!paths.isEmpty()) {
                TB_LOG_DEBUG(LogModule::Clipboard,
                    "Detected {} file(s) in clipboard", paths.size());
                return Result<ClipboardItem>::ok(
                    ClipboardItem::fromFiles(paths));
            }
        }

        // 检查 HTML 文本
        QString htmlText = QString::fromUtf8(mimeData->data("text/html"));
        if (!htmlText.isEmpty()) {
            TB_LOG_DEBUG(LogModule::Clipboard, "Detected HTML text");
            return Result<ClipboardItem>::ok(
                ClipboardItem::fromText(htmlText, true));
        }

        // 检查纯文本
        QString plainText = mimeData->text();
        if (!plainText.isEmpty()) {
            TB_LOG_DEBUG(LogModule::Clipboard, "Detected text, length={}", plainText.length());
            return Result<ClipboardItem>::ok(
                ClipboardItem::fromText(plainText, false));
        }

        return Result<ClipboardItem>::err(
            TB_MAKE_ERROR(ErrorCode::ClipboardEmpty, 
                "Clipboard contains no supported format"));

    } catch (const std::exception& e) {
        TB_LOG_ERROR(LogModule::Clipboard, "Exception: {}", e.what());
        return Result<ClipboardItem>::err(
            TB_MAKE_ERROR(ErrorCode::ClipboardReadError, e.what()));
    }
}
```

```sql
-- src/services/clipboard/schema.sql

-- 主表：剪贴板条目
CREATE TABLE IF NOT EXISTS clipboard_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    type INTEGER NOT NULL,                 -- 0=Text, 1=Image, 2=Files
    content BLOB NOT NULL,                 -- 序列化的内容
    preview TEXT,                          -- 预览文本（≤200字符）
    created_at DATETIME NOT NULL,          -- 创建时间
    accessed_at DATETIME NOT NULL,         -- 最后访问时间
    pinned INTEGER DEFAULT 0,              -- 是否置顶
    access_count INTEGER DEFAULT 0,        -- 被恢复次数
    size_bytes INTEGER DEFAULT 0           -- 估算大小
);

-- 索引：加快查询
CREATE INDEX IF NOT EXISTS idx_created_at ON clipboard_items(created_at DESC);
CREATE INDEX IF NOT EXISTS idx_accessed_at ON clipboard_items(accessed_at DESC);
CREATE INDEX IF NOT EXISTS idx_type ON clipboard_items(type);
CREATE INDEX IF NOT EXISTS idx_pinned ON clipboard_items(pinned DESC, created_at DESC);

-- 标签表
CREATE TABLE IF NOT EXISTS clipboard_tags (
    item_id INTEGER NOT NULL,
    tag TEXT NOT NULL,
    FOREIGN KEY (item_id) REFERENCES clipboard_items(id) ON DELETE CASCADE,
    PRIMARY KEY (item_id, tag)
);

CREATE INDEX IF NOT EXISTS idx_tag_name ON clipboard_tags(tag);

-- 全文搜索虚表（FTS5）
CREATE VIRTUAL TABLE IF NOT EXISTS clipboard_search
USING fts5(
    item_id UNINDEXED,
    preview,
    content=clipboard_items,
    content_rowid=id
);

-- 创建触发器保持 FTS 索引最新
CREATE TRIGGER IF NOT EXISTS clipboard_items_ai AFTER INSERT ON clipboard_items BEGIN
    INSERT INTO clipboard_search(rowid, item_id, preview)
    VALUES (new.id, new.id, new.preview);
END;

CREATE TRIGGER IF NOT EXISTS clipboard_items_ad AFTER DELETE ON clipboard_items BEGIN
    DELETE FROM clipboard_search WHERE item_id = old.id;
END;
```

### 7.3 内容序列化/反序列化

```cpp
// 使用 Qt JSON 序列化

QJsonObject serializeClipboardItem(const ClipboardItem& item) {
    QJsonObject obj;
    obj["type"] = static_cast<int>(item.type);
    obj["preview"] = item.preview;
    obj["created_at"] = item.createdAt.toSecsSinceEpoch();
    obj["accessed_at"] = item.accessedAt.toSecsSinceEpoch();
    obj["pinned"] = item.pinned;
    obj["access_count"] = item.accessCount;
    obj["tags"] = QJsonArray::fromStringList(item.tags);

    switch (item.type) {
        case ClipboardContentType::Text: {
            const auto& text = std::get<ClipboardTextContent>(item.content);
            QJsonObject content;
            content["text"] = text.text;
            content["plainText"] = text.plainText;
            content["isHtml"] = text.isHtml;
            obj["content"] = content;
            break;
        }
        case ClipboardContentType::Image: {
            const auto& img = std::get<ClipboardImageContent>(item.content);
            QJsonObject content;
            content["pngData"] = QString::fromLatin1(img.pngData.toBase64());
            content["width"] = img.width;
            content["height"] = img.height;
            obj["content"] = content;
            break;
        }
        case ClipboardContentType::Files: {
            const auto& files = std::get<ClipboardFilesContent>(item.content);
            obj["content"] = QJsonArray::fromStringList(files.paths);
            break;
        }
        default:
            break;
    }

    return obj;
}
```

```qml
        thumbnail: model.thumbnail
        createdAt: model.createdAt
        pinned: model.pinned
        tags: model.tags

        onRestoreClicked: root.viewModel.restoreItem(model.itemId)
        onDeleteClicked: root.viewModel.deleteItem(model.itemId)
        onTogglePinClicked: root.viewModel.togglePin(model.itemId)
        onEditTagsClicked: tagEditorDialog.open(model.itemId, model.tags)
    }

    onAtYEndChanged: {
        if (atYEnd && !root.viewModel.isLoading) {
            root.viewModel.loadMore()
        }
    }

    // ── 标签编辑对话框 ──────────────────────────────────────────────
    TagEditorDialog {
        id: tagEditorDialog
        availableTags: root.viewModel.availableTags
        onConfirmed: (itemId, tags) => root.viewModel.editTags(itemId, tags)
    }

    // ── 空状态提示 ──────────────────────────────────────────────────
    Text {
        anchors.centerIn: parent
        visible: root.count === 0
        text: qsTr("暂无历史记录")
        color: Theme.textSecondary
        font.pixelSize: 14
    }

    BusyIndicator {
        anchors.centerIn: parent
        visible: root.viewModel.isLoading
    }
}
```

## 13. 性能优化

### 13.1 数据库查询优化

- **FTS5 全文搜索**：使用 SQLite 虚表加速关键词查询
- **分页加载**：避免一次加载过多记录
- **连接池**：复用数据库连接
- **异步 I/O**：使用 ThreadPool 避免阻塞 UI

### 13.2 内存优化

- **图片压缩**：PNG 格式存储，使用质量=85 的 JPEG 缩略图
- **延迟加载**：缩略图在 ListView 显示时生成
- **LRU 清理**：超过 maxSize 时自动删除最老的记录

### 13.3 缓存策略

- **预留热点数据缓存**：最近 50 条记录在内存中
- **标签缓存**：避免频繁查询所有标签

## 14. 已知问题与限制

| 问题 | 描述 | 解决方案 |
|------|------|--------|
| 图片超大 | > 5MB 的图片不记录 | 可配置 imageMaxSizeKB |
| 跨进程剪贴板 | Windows 某些进程无法读取剪贴板 | 使用 Win32 API 备用方案（v2.0+） |
| 密码泄露 | 某些密码管理器的剪贴板数据被记录 | blacklistApps 配置黑名单应用 |
| 性能（超大历史） | > 1000 条记录查询缓慢 | 实施自动清理策略 |
