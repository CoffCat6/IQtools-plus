## 1. 功能概述

### 1.1 核心职责

翻译模块负责：

- **多引擎支持**：有道翻译、DeepL、GPT（可扩展）
- **热切换机制**：用户可实时切换翻译引擎，自动降级
- **划词翻译**：全局快捷键触发，非侵入式
- **智能缓存**：LRU 内存缓存 + SQLite 持久化，避免重复请求
- **语言检测**：自动识别源语言
- **历史管理**：保存翻译历史，支持快速复用
- **离线支持**（预留）：缓存命中时无需网络

### 1.2 性能目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 缓存命中响应 | < 50ms | 从缓存返回结果 |
| 首次翻译延迟 | < 2s | 单个请求的总耗时（含网络） |
| 缓存容量 | 1000 条 | 内存 + 数据库 |
| 平均响应时间 | < 500ms | 排除网络延迟 |
| 同时并发请求 | 10 个 | 线程池容量 |

### 1.3 支持的语言

```
英文 (en)         日文 (ja)         韩文 (ko)
中文简体 (zh-CN)  繁体 (zh-TW)      西班牙文 (es)
法文 (fr)         德文 (de)         俄文 (ru)
葡文 (pt)         阿拉伯文 (ar)     泰文 (th)
越南文 (vi)       印尼文 (id)       自动检测 (auto)
```

## 2. 需求分析

### 2.1 功能需求

#### FR1：基础翻译
- 支持文本翻译（单行、多行）
- 支持指定源语言和目标语言
- 自动检测源语言（auto）
- 返回翻译结果 + 备选方案

#### FR2：多引擎支持
- 集成有道翻译（免费）
- 集成 DeepL API（付费 + 免费 API）
- 预留 GPT 接口（v2.0+）
- 支持引擎优先级和自动降级

#### FR3：缓存机制
- 内存 LRU 缓存（1000 条）
- SQLite 持久化缓存（无限制）
- 缓存命中率统计

#### FR4：划词翻译
- 全局快捷键 (Ctrl+Shift+X)
- 监听鼠标选择事件
- 弹窗/侧边栏展示结果

#### FR5：历史与管理
- 保存翻译历史
- 按时间/频率排序
- 支持导出历史记录

### 2.2 非功能需求

| 类别 | 需求 |
|------|------|
| 性能 | 缓存命中 < 50ms，网络请求 < 2s |
| 可用性 | 支持快捷键操作、手动切换引擎 |
| 可靠性 | 引擎故障自动降级，无单点故障 |
| 可扩展性 | 易于添加新引擎（插件接口） |
| 安全 | API Key 加密存储，无明文日志 |
| 跨平台 | Windows / macOS / Linux |

### 3.2 数据流（正常翻译流程）

```
用户输入文本 (QML)
    ↓
[TranslateViewModel::translate()]
    ↓
[TranslateService::translateAsync()] 
    ↓
查询 LRU 缓存 ──缓存命中──→ 返回结果 → 回调 ViewModel → 更新 UI
    ↓ (未命中)
查询数据库缓存 ──数据库命中──→ 更新内存缓存 → 返回结果
    ↓ (未命中)
优先级队列：
    1. 尝试首选引擎 (如有道)
    2. 引擎不可用 → 降级到备选 (DeepL)
    3. 所有引擎均失败 → 返回错误
    ↓
发起 HTTP 请求 (异步，线程池)
    ↓
解析响应 + 写入缓存
    ↓
回调 ViewModel (通过 EventBus)
    ↓
ViewModel 更新 UI
```

### 3.3 线程模型

```
主线程 (Qt GUI Thread):
  - ViewModel 的 slot/signal
  - QML UI 操作
  - 禁止阻塞 > 16ms

网络线程 (ThreadPool):
  - TranslateService 的异步翻译
  - HTTP 请求 (libcurl / Qt Network)
  - 缓存查询、写入

数据库线程 (SQLite 内部):
  - TranslateCache 的 SQLite 操作
  - 自动加锁，不需要额外同步

全局快捷键线程（系统级）:
  - 监听鼠标选择
  - 触发划词翻译
```

## 5. 数据结构定义

### 5.1 翻译请求/结果

```cpp
// src/services/translate/TranslateTypes.h

#pragma once
#include <QString>
#include <QStringList>
#include <QDateTime>
#include <QVariantMap>
#include <cstdint>

// ── 翻译请求 ────────────────────────────────────────────────────────────
struct TranslateRequest {
    QString text;               // 源文本
    QString fromLang;           // 源语言（"auto" 自动检测）
    QString toLang;             // 目标语言
    QString engineId;           // 指定引擎（空 = 使用当前默认）
    int     timeoutMs = 10000;  // 请求超时（毫秒）
    
    // 工厂方法
    [[nodiscard]] static TranslateRequest autoDetect(
        const QString& text, 
        const QString& toLang = QStringLiteral("zh-CN"));
};

// ── 翻译结果 ────────────────────────────────────────────────────────────
struct TranslateResult {
    QString       translatedText;     // 主翻译结果
    QString       detectedLang;       // 自动检测到的源语言
    QString       engineId;           // 实际使用的引擎
    QStringList   alternatives;       // 备选翻译（可选）
    int           latencyMs{0};       // 请求耗时
    QDateTime     timestamp;          // 结果时间戳
    QVariantMap   metadata;           // 引擎特定元数据
    
    [[nodiscard]] bool isEmpty() const noexcept {
        return translatedText.isEmpty();
    }
};

// ── 缓存项 ──────────────────────────────────────────────────────────────
struct TranslateCacheEntry {
    uint64_t      id{0};
    QString       sourceText;
    QString       fromLang;
    QString       toLang;
    QString       translatedText;
    QString       engineId;
    QDateTime     createdAt;
    QDateTime     accessedAt;          // LRU：最后访问时间
    int           accessCount{0};      // 使用频率
    
    // 索引键（用于快速查询）
    [[nodiscard]] QString cacheKey() const noexcept {
        // 格式：fromLang|toLang|sourceText (hash)
        return fromLang + QChar('|') + toLang + QChar('|') + sourceText;
    }
};

// ── 语言配置 ────────────────────────────────────────────────────────────
struct Language {
    QString code;        // "en", "zh-CN", etc.
    QString name;        // "English", "简体中文"
    QString nativeName;  // "English", "中文"
    QString region;      // "US", "CN" （可选）
};

// ── 统计信息 ────────────────────────────────────────────────────────────
struct TranslateStatistics {
    int     totalRequests{0};
    int     cacheHits{0};
    int     cacheMisses{0};
    int     failedRequests{0};
    double  avgLatencyMs{0.0};
    
    [[nodiscard]] double hitRate() const noexcept {
        return totalRequests > 0 
            ? static_cast<double>(cacheHits) / totalRequests 
            : 0.0;
    }
};
```

## 6. 翻译引擎接口

### 6.1 ITranslateEngine 抽象接口

```cpp
// src/services/translate/ITranslateEngine.h

#pragma once
#include <QString>
#include <QStringList>
#include <functional>
#include <memory>
#include "core/error/Error.h"
#include "TranslateTypes.h"

/**
 * @brief 翻译引擎抽象接口
 *
 * 所有具体翻译引擎（有道、DeepL、GPT 等）都必须继承此接口。
 * 支持同步和异步翻译两种模式。
 *
 * 线程安全：
 * - 可从任意线程调用
 * - 异步方法的回调在 ThreadPool 线程中执行
 */
class ITranslateEngine {
public:
    virtual ~ITranslateEngine() = default;

    // ── 元信息 ────────────────────────────────────────────────────────────
    /**
     * @brief 获取引擎 ID
     * @return "youdao", "deepl", "gpt" 等
     */
    [[nodiscard]] virtual QString engineId() const noexcept = 0;

    /**
     * @brief 获取引擎显示名称
     * @return "有道翻译", "DeepL", "GPT-3.5"
     */
    [[nodiscard]] virtual QString engineName() const noexcept = 0;

    /**
     * @brief 获取支持的语言列表
     */
    [[nodiscard]] virtual QStringList supportedLanguages() const noexcept = 0;

    /**
     * @brief 检查引擎当前是否可用
     *
     * 检查因素：
     * - API Key 是否有效
     * - 网络连接状态
     * - 是否达到速率限制
     *
     * @return true 如果可用
     */
    [[nodiscard]] virtual bool isAvailable() const noexcept = 0;

    /**
     * @brief 检查特定语言对是否支持
     */
    [[nodiscard]] virtual bool supportsLanguagePair(
        const QString& fromLang,
        const QString& toLang) const noexcept = 0;

    // ── 异步翻译（异步模式，推荐） ────────────────────────────────────────
    using TranslateCallback = std::function<void(Result<TranslateResult>)>;

    /**
     * @brief 异步翻译
     *
     * 在线程池中执行，避免阻塞调用方。
     * 完成后通过 callback 返回结果。
     *
     * @param request   翻译请求
     * @param callback  完成回调（在 ThreadPool 线程中调用）
     *
     * @note 若请求在 callback 调用前被取消，则 callback 不会被调用
     * @note 回调中的错误应由调用方处理
     */
    virtual void translateAsync(
        const TranslateRequest& request,
        TranslateCallback callback) = 0;

    /**
     * @brief 取消异步翻译
     *
     * 尝试取消未完成的请求。
     * 若请求已完成或不存在，此操作无效。
     *
     * @param requestId 请求 ID（可选，若为空则取消最新请求）
     */
    virtual void cancelTranslate(const QString& requestId = {}) {}

    // ── 同步翻译（阻塞模式，仅供测试/命令行） ──────────────────────────────
    /**
     * @brief 同步翻译（阻塞）
     *
     * @param request 翻译请求
     * @return 翻译结果或错误
     *
     * @warning 不要在主线程（UI 线程）调用此方法
     */
    [[nodiscard]] virtual Result<TranslateResult>
    translateSync(const TranslateRequest& request) = 0;

    // ── 配置 ────────────────────────────────────────────────────────────────
    /**
     * @brief 配置引擎参数
     *
     * @param config 配置字典，格式由各引擎定义
     *              常见键：
     *              - "apiKey": 引擎 API Key
     *              - "appId": 应用 ID
     *              - "appSecret": 应用密钥
     *              - "freeApi": 是否使用免费 API 端点
     *
     * @return 配置是否成功
     */
    [[nodiscard]] virtual Result<void>
    configure(const QVariantMap& config) noexcept = 0;

    /**
     * @brief 获取当前配置
     */
    [[nodiscard]] virtual QVariantMap getConfiguration() const noexcept = 0;

    // ── 语言检测（可选） ───────────────────────────────────────────────────
    /**
     * @brief 检测文本语言
     *
     * @param text 源文本
     * @return 语言代码（如 "en", "zh-CN"）或错误
     *
     * @note 默认实现返回 "auto"（不支持检测）
     */
    [[nodiscard]] virtual Result<QString>
    detectLanguage(const QString& text) const noexcept {
        return Result<QString>::ok(QStringLiteral("auto"));
    }

    // ── 配额与限制 ──────────────────────────────────────────────────────────
    /**
     * @brief 获取当前速率限制状态
     *
     * @return {maxRequests, remainingRequests, resetTimeMs}
     */
```

## 8. 缓存系统设计

### 8.1 TranslateCache 接口

```cpp
// src/services/translate/TranslateCache.h

#pragma once
#include <QString>
#include <QList>
#include <QReadWriteLock>
#include <memory>
#include "TranslateTypes.h"
#include "core/error/Error.h"

class QSqlDatabase;

/**
 * @brief 双层缓存：LRU 内存 + SQLite 持久化
 *
 * 工作流程：
 * 1. 查询 LRU（O(1)）
 * 2. 未命中 → 查询 SQLite（O(log n)）
 * 3. SQLite 命中 → 更新 LRU 并返回
 * 4. 均未命中 → 返回未命中
 * 5. 新条目 → 同时写入 LRU 和 SQLite
 *
 * 线程安全：
 * - 使用 QReadWriteLock 保护内存 LRU
 * - SQLite 内部自动加锁
 */
class TranslateCache : public QObject {
    Q_OBJECT

public:
    explicit TranslateCache(int maxMemorySize = 1000, QObject* parent = nullptr);
    ~TranslateCache() override;

    // ── 初始化 ──────────────────────────────────────────────────────────
    /**
     * @brief 打开缓存数据库
     *
     * @param dbPath 数据库文件路径
     * @return 打开是否成功
     */
    [[nodiscard]] Result<void>
    openDatabase(const QString& dbPath) noexcept;

    /**
     * @brief 关闭数据库
     */
    void closeDatabase() noexcept;

    /**
     * @brief 检查数据库连接状态
     */
    [[nodiscard]] bool isDatabaseOpen() const noexcept;

    // ── 查询 ────────────────────────────────────────────────────────────
    /**
     * @brief 查询缓存条目
     *
     * @param sourceText 源文本
     * @param fromLang 源语言
     * @param toLang 目标语言
     * @return 缓存条目或空（未命中）
     */
    [[nodiscard]] Result<TranslateCacheEntry>
    find(const QString& sourceText,
         const QString& fromLang,
         const QString& toLang) noexcept;

    /**
     * @brief 按 ID 查询
     */
    [[nodiscard]] Result<TranslateCacheEntry>
    findById(uint64_t id) noexcept;

    // ── 写入 ────────────────────────────────────────────────────────────
    /**
     * @brief 插入或更新缓存条目
     *
     * @param entry 缓存条目
     * @return 新的条目 ID
     */
    [[nodiscard]] Result<uint64_t>
    insert(const TranslateCacheEntry& entry) noexcept;

    // ── 删除 ────────────────────────────────────────────────────────────
    /**
     * @brief 删除条目
     */
    [[nodiscard]] Result<void>
    deleteEntry(uint64_t id) noexcept;

    /**
     * @brief 批量删除
     */
    [[nodiscard]] Result<int>
    deleteEntries(const QList<uint64_t>& ids) noexcept;

    /**
     * @brief 清空 LRU 缓存
     */
    void clearMemory() noexcept;

    /**
     * @brief 清空数据库缓存
     */
    [[nodiscard]] Result<void>
    clearDatabase() noexcept;

    // ── 统计 ────────────────────────────────────────────────────────────
    /**
     * @brief 获取缓存统计
     */
    [[nodiscard]] TranslateStatistics getStatistics() const noexcept;

    /**
     * @brief 获取历史记录
     */
    [[nodiscard]] Result<QList<TranslateCacheEntry>>
    getHistory(int limit = 50, int offset = 0) noexcept;

    /**
     * @brief 清理过期数据（超过 maxDbSize）
     */
    [[nodiscard]] Result<int>
    cleanup(int maxDbSize = 10000) noexcept;

    /**
     * @brief LRU 命中率
     */
    [[nodiscard]] double getHitRate() const noexcept;

signals:
    /**
     * @brief 缓存统计已更新
     */
    void statisticsChanged(const TranslateStatistics& stats);

private:
    Result<void> initializeDatabase() noexcept;
    Result<TranslateCacheEntry> queryDatabase(
        const QString& sourceText,
        const QString& fromLang,
        const QString& toLang) noexcept;

    // LRU 缓存（内存）
    mutable QReadWriteLock m_memoryLock;
    struct LRUNode {
        TranslateCacheEntry entry;
        LRUNode* prev{nullptr};
        LRUNode* next{nullptr};
    };
    std::map<QString, std::unique_ptr<LRUNode>> m_lruMap;
    LRUNode* m_lruHead{nullptr};
    LRUNode* m_lruTail{nullptr};
    int m_maxMemorySize;

    // SQLite 数据库
    std::unique_ptr<QSqlDatabase> m_db;
    bool m_dbInitialized{false};

    // 统计
    mutable QReadWriteLock m_statsLock;
    TranslateStatistics m_statistics;
};
```

## 12. 多线程与异步

### 12.1 异步翻译流程

```cpp
// TranslateService::translateAsync 的实现概要

void TranslateService::translateAsync(
    const TranslateRequest& request,
    TranslateCallback callback) noexcept
{
    // 1. 查询 LRU 缓存（主线程，快速）
    if (auto cached = m_cache->find(request.text, request.fromLang, request.toLang)) {
        // 缓存命中
        QMetaObject::invokeMethod(this, [this, cached, callback]() {
            callback(Result<TranslateResult>::ok(cached.value()));
        }, Qt::QueuedConnection);
        return;
    }

    // 2. 异步查询数据库并翻译（线程池）
    auto task = [this, request, callback]() {
        // 2a. 查询 SQLite
        if (auto dbCached = m_cache->find(...)) {
            QMetaObject::invokeMethod(this, [this, dbCached, callback]() {
                callback(Result<TranslateResult>::ok(dbCached.value()));
            }, Qt::QueuedConnection);
            return;
        }

        // 2b. 调用引擎翻译
        auto engine = selectBestEngine(request.engineId);
        auto result = engine->translateSync(request);

        // 2c. 写入缓存
        if (result.isOk()) {
            m_cache->insert(result.value());
        }

        // 2d. 回调（返回主线程）
        QMetaObject::invokeMethod(this, [callback, result]() {
            callback(result);
        }, Qt::QueuedConnection);
    };

    m_threadPool->enqueue(task);
}
```

### 13.2 错误恢复策略

```
翻译请求失败处理链：

1. 当前引擎失败
   ↓
   尝试备选引擎（按优先级）
   ├─ 有道 → DeepL → GPT
   └─ 返回第一个成功的结果

2. 所有引擎均失败
   ↓
   返回缓存结果（如果存在）
   └─ 若无缓存，返回错误

3. 网络超时
   ↓
   重试 N 次（可配置）
   ├─ 成功 → 返回
   └─ 最终失败 → 返回错误

4. 认证失败（API Key 无效）
   ↓
   记录日志，禁用该引擎
   ↓
   自动降级到备选引擎
```
