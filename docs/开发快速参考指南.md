## 🏗️ 架构速览

### 分层架构

```
┌─────────────────────────────────────────┐
│         QML UI Layer                    │  ← 纯界面，不含业务逻辑
│    (Pages / Components / Dialogs)       │
└─────────────────┬───────────────────────┘
                  │ Q_PROPERTY / signals
┌─────────────────▼───────────────────────┐
│       ViewModel Layer (QObject)          │  ← 数据绑定、业务逻辑
│  (ScreenshotVM / TranslateVM / etc)     │
└─────────────────┬───────────────────────┘
                  │ C++ 函数调用
┌─────────────────▼───────────────────────┐
│        Service Layer (纯 C++)            │  ← 业务逻辑，无 UI 依赖
│   (ScreenshotService / TranslateService)│
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│      Core Infrastructure                │  ← 跨模块基础设施
│  (Logger / EventBus / ConfigManager)    │
└─────────────────────────────────────────┘
```

### 关键设计模式

| 模式 | 应用场景 | 示例 |
|------|----------|------|
| MVVM | QML ↔ ViewModel ↔ Service | 所有页面 |
| 依赖注入 | 服务生命周期管理 | AppContext |
| 观察者 | 状态变更通知 | ViewModel signals |
| 策略模式 | 多实现切换 | 翻译引擎 |
| Result<T> | 错误处理 | 所有 Service 返回值 |

## 🎨 Soft UI 设计规范速览

### 核心原则

```
✅ 圆润优先      → 所有元素 radius ≥ 8px
✅ 阴影叠加      → 用阴影表达层级，不用边框
✅ 低饱和度      → 避免鲜艳纯色
✅ 白空间充足    → 间距基数 8px
✅ 微妙动画      → 时长 200-400ms，缓动 Easing.InOutQuad
❌ 禁止硬边框    → 全部用阴影
❌ 禁止零圆角    → 最小 8px
❌ 禁止纯黑色    → 使用 #1A1A1A 或更浅
```

### 设计代币

```qml
// 色彩
Theme.colorPrimary          // 主色 #5B63F5
Theme.colorSuccess          // 成功 #2DA44E
Theme.colorError            // 错误 #DC3545
Theme.colorWarning          // 警告 #FF9800

// 间距（8px 栅格）
8px / 16px / 24px / 32px / 48px

// 圆角
8px (小)  / 12px (中)  / 16px (大)  / 24px (特大)

// 阴影（5 个等级）
Shadow-XS / Shadow-S / Shadow-M / Shadow-L / Shadow-XL

// 字体
Theme.fontHeading1 / fontHeading2 / fontHeading3
Theme.fontBody1 / fontBody2 / fontCaption
```

### 常用组件

```qml
// 按钮
Button {
    background: Rectangle { radius: 12; color: Theme.colorPrimary }
}

// 卡片
Rectangle {
    radius: 16
    color: Theme.colorBackground
    layer.effect: DropShadow { blur: 8; yOffset: 2 }
}

// 输入框
TextField {
    background: Rectangle { radius: 12; border.color: Theme.colorBorder }
}

// 开关
Switch {
    background: Rectangle { radius: 12; color: checked ? Theme.colorPrimary : Theme.colorBorder }
}
```

## 📝 编码规范速览

### C++ 规范

```cpp
// 命名规范
class MyService { };           // 类：PascalCase
void myFunction() { }          // 函数：camelCase
int m_memberVariable;          // 成员变量：m_ 前缀
const int CONSTANT_VALUE = 0;  // 常量：UPPER_SNAKE_CASE

// 错误处理（禁用异常）
Result<QString> getValue() {
    if (error) {
        return Result<QString>::error(Error(ErrorCode::InvalidArgument, "msg"));
    }
    return Result<QString>::ok(value);
}

// 多线程安全
QMutex m_mutex;
QMutexLocker locker(&m_mutex);

// 信号槽
connect(source, &Source::signal, this, &Receiver::slot);
```

### QML 规范

```qml
// 命名规范
Rectangle { }                  // 类型：PascalCase
property string myProperty     // 属性：camelCase
signal mySignal()              // 信号：camelCase
function myFunction() { }      // 函数：camelCase

// 布局

    anchors.fill: parent
    anchors.margins: 16
    spacing: 12
}

// 绑定
Text {
    text: viewModel.title
    color: Theme.colorText
    font: Theme.fontBody1

// 动画
Behavior on opacity {
    NumberAnimation { duration: 200; easing.type: Easing.InOutQuad }
}
```

## 🔧 常见任务

### 添加新的设置项

1. **在 SettingsRegistry 中注册**
   ```cpp
   SettingItem item{
       .id = "my_setting",
       .category = "通用",
       .label = "我的设置",
       .type = "switch"
   };
   registry->registerSetting("plugin_id", item);
   ```

2. **在 SettingsViewModel 中处理**
   ```cpp
   void SettingsViewModel::updateSetting(const QString& id, const QVariant& value) {
       m_pendingChanges[id] = value;
   }
   ```

3. **在 QML 中显示**
   ```qml
   SettingItem {
       label: "我的设置"
       controlComponent: SettingSwitch {
           checked: settingItem.currentValue
           onToggled: settingsVM.updateSetting(settingItem.id, checked)
       }
   }
   ```

### 添加新的 QML 页面

1. **创建页面文件** `src/ui/pages/MyPage.qml`
   ```qml
   import QtQuick
   import "../components"
   import "../theme"
   
   Page {
       title: "我的页面"
       background: Rectangle { color: Theme.colorPageBackground }
   }
   ```

2. **在 main.qml 中注册**
   ```qml
   StackView {
       MyPage { }
   }
   ```

3. **创建对应的 ViewModel** `src/viewmodels/MyViewModel.h`
   ```cpp
   class MyViewModel : public QObject {
       Q_OBJECT
       // ...
   };
   ```

### 添加新的 Service

1. **定义接口** `src/services/MyService.h`
   ```cpp
   class MyService {
   public:
       Result<QString> doSomething();
   };
   ```

2. **实现服务** `src/services/MyService.cpp`
   ```cpp
   Result<QString> MyService::doSomething() {
       // 实现逻辑
       return Result<QString>::ok("result");
   }
   ```

3. **在 AppContext 中注册**
   ```cpp
   auto myService = new MyService();
   m_services["MyService"] = myService;
   ```

## 📊 性能优化建议

### QML 优化

```qml
// ❌ 不好：每次都创建
Repeater {
    model: largeList
    delegate: Rectangle { /* 复杂组件 */ }
}

// ✅ 好：使用 Loader 延迟加载
Repeater {
    model: largeList
    delegate: Loader {
        asynchronous: true
        sourceComponent: complexComponent
    }
}

// ✅ 好：使用 TableView 虚拟化
TableView {
    model: largeList
    delegate: Rectangle { /* 简单组件 */ }
}
```

### C++ 优化

```cpp
// ❌ 不好：阻塞 UI 线程
void MyService::loadData() {
    // 耗时操作
    QThread::sleep(5000);
}

// ✅ 好：使用线程池
void MyService::loadData() {
    QtConcurrent::run([this]() {
        // 耗时操作
        emit dataLoaded(result);
    });
}
```

## 🐛 调试技巧

### 启用调试日志

```cpp
// 在 main.cpp 中
Logger::setLevel(LogLevel::Debug);
Logger::addSink(new ConsoleSink());
Logger::addSink(new FileSink("logs/app.log"));

// 在代码中
LOG_DEBUG("Debug message: {}", value);
LOG_INFO("Info message");
LOG_WARN("Warning message");
LOG_ERROR("Error message");
```

### Qt Creator 调试

```
1. 设置断点：点击行号左侧
2. 启动调试：F5 或 Debug > Start Debugging
3. 单步执行：F10（单步）/ F11（进入）
4. 查看变量：Locals 窗口
5. 计算表达式：Debug > Evaluate Expression
```

### QML 调试

```qml
// 打印日志
console.log("Debug:", value)
console.warn("Warning:", value)
console.error("Error:", value)

// 使用 Qt.binding 调试绑定
property var debugBinding: Qt.binding(function() {
    console.log("Binding updated:", value)
    return value
})
```
