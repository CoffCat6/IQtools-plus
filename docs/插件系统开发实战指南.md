## 1. 插件系统全景

### 1.1 插件系统做什么

```
没有插件系统：
  所有功能（翻译、截图、OCR、二维码...）都编译进主程序。
  添加新功能 = 修改主程序代码 + 重新编译 + 重新发布。

有插件系统：
  核心功能在主程序中，扩展功能以插件形式存在。
  添加新功能 = 开发一个插件 .dll → 放到 plugins/ 目录 → 主程序自动加载。
```

## 2. Qt 插件机制基础

### 2.1 Qt 插件的三要素

Qt 的插件系统基于三个关键宏/类：

```
宿主端（主程序）                      插件端（.dll）
─────────────────                    ─────────────────
                                     Q_INTERFACES(IPlugin)
Q_DECLARE_INTERFACE(IPlugin, IID)    Q_PLUGIN_METADATA(IID IID FILE "metadata.json")
                                     ↑
QPluginLoader loader(path);          类实现 IPlugin 的所有虚函数
QObject* obj = loader.instance();
IPlugin* plugin = qobject_cast<IPlugin*>(obj);
```

### 2.2 静态链接 vs 动态加载

本项目支持两种插件模式：

```
模式 A：静态链接（内置插件）
  插件代码在 plugins/ 目录下，与主程序一起编译。
  适合：官方插件、必须内置的功能。
  实现：CMake add_subdirectory + 静态库链接。

模式 B：动态加载（外部插件）
  插件编译为独立的 .dll/.so，放在 plugins/ 运行时目录。
  适合：第三方插件、可选功能。
  实现：QPluginLoader 在运行时加载。

本指南两种都会讲解。先实现动态加载（更通用），再讲解静态链接的适配。
```

### 3.1 Step 1：定义插件元信息

**新建文件：** `src/core/plugin/PluginMetadata.h`

```cpp
// src/core/plugin/PluginMetadata.h
#pragma once

#include <QString>
#include <QStringList>
#include <QVersionNumber>
#include <QJsonObject>
#include <QJsonArray>

/// 插件元信息
///
/// 描述一个插件的身份、版本、依赖和权限要求。
/// 从插件的 JSON 元数据文件中解析。
struct PluginMetadata {
    QString        id;              // 全局唯一标识，如 "com.iqtools.ocr"
    QString        name;            // 显示名称，如 "OCR 文字识别"
    QString        description;     // 简短描述
    QString        author;          // 作者
    QVersionNumber version;         // 插件版本
    QVersionNumber minHostVersion;  // 要求的最低主程序版本
    QStringList    dependencies;    // 依赖的其他插件 ID
    QString        iconPath;        // 图标资源路径（qrc 或文件路径）
    QStringList    permissions;     // 所需权限列表
    QString        mainQml;         // 插件主 QML 组件路径（可选）
    bool           enabled{true};   // 用户是否启用

    /// 从 JSON 对象解析
    [[nodiscard]] static PluginMetadata fromJson(const QJsonObject& obj) {
        PluginMetadata m;
        m.id              = obj[QStringLiteral("id")].toString();
        m.name            = obj[QStringLiteral("name")].toString();
        m.description     = obj[QStringLiteral("description")].toString();
        m.author          = obj[QStringLiteral("author")].toString();
        m.version         = QVersionNumber::fromString(
                                obj[QStringLiteral("version")].toString());
        m.minHostVersion  = QVersionNumber::fromString(
                                obj[QStringLiteral("minHostVersion")].toString());
        m.iconPath        = obj[QStringLiteral("icon")].toString();
        m.mainQml         = obj[QStringLiteral("mainQml")].toString();

        // 解析依赖数组
        const QJsonArray deps = obj[QStringLiteral("dependencies")].toArray();
        for (const auto& d : deps) {
            m.dependencies.append(d.toString());
        }

        // 解析权限数组
        const QJsonArray perms = obj[QStringLiteral("permissions")].toArray();
        for (const auto& p : perms) {
            m.permissions.append(p.toString());
        }

        return m;
    }

    /// 序列化为 JSON
    [[nodiscard]] QJsonObject toJson() const {
        QJsonObject obj;
        obj[QStringLiteral("id")]              = id;
        obj[QStringLiteral("name")]            = name;
        obj[QStringLiteral("description")]     = description;
        obj[QStringLiteral("author")]          = author;
        obj[QStringLiteral("version")]         = version.toString();
        obj[QStringLiteral("minHostVersion")]  = minHostVersion.toString();
        obj[QStringLiteral("icon")]            = iconPath;
        obj[QStringLiteral("mainQml")]         = mainQml;

        QJsonArray deps;
        for (const auto& d : dependencies) deps.append(d);
        obj[QStringLiteral("dependencies")] = deps;

        QJsonArray perms;
        for (const auto& p : permissions) perms.append(p);
        obj[QStringLiteral("permissions")] = perms;

        return obj;
    }
};
```

### 3.2 Step 2：定义权限枚举

**新建文件：** `src/core/plugin/PluginPermission.h`

```cpp
// src/core/plugin/PluginPermission.h
#pragma once

#include <cstdint>
#include <QString>
#include <QSet>

/// 插件权限枚举
///
/// 每个插件在 metadata 中声明所需权限，宿主在加载时校验。
/// 未声明的权限，插件不应尝试使用。
enum class PluginPermission : uint32_t {
    ScreenshotRead   = 1 << 0,   // 读取截图数据
    ClipboardRead    = 1 << 1,   // 读取剪贴板
    ClipboardWrite   = 1 << 2,   // 写入剪贴板
    FileSystemRead   = 1 << 3,   // 读取文件
    FileSystemWrite  = 1 << 4,   // 写入文件
    NetworkAccess    = 1 << 5,   // 网络请求
    HotkeyRegister   = 1 << 6,   // 注册全局快捷键
    AI               = 1 << 7,   // 使用 AI 服务
};

/// 权限字符串 ↔ 枚举转换
namespace PluginPermUtil {

inline PluginPermission fromString(const QString& str) {
    static const QMap<QString, PluginPermission> mapping = {
        {QStringLiteral("screenshot.read"),   PluginPermission::ScreenshotRead},
        {QStringLiteral("clipboard.read"),    PluginPermission::ClipboardRead},
        {QStringLiteral("clipboard.write"),   PluginPermission::ClipboardWrite},
        {QStringLiteral("filesystem.read"),   PluginPermission::FileSystemRead},
        {QStringLiteral("filesystem.write"),  PluginPermission::FileSystemWrite},
        {QStringLiteral("network"),           PluginPermission::NetworkAccess},
        {QStringLiteral("hotkey"),            PluginPermission::HotkeyRegister},
        {QStringLiteral("ai"),                PluginPermission::AI},
    };
    return mapping.value(str, static_cast<PluginPermission>(0));
}

inline QSet<PluginPermission> parsePermissionList(const QStringList& list) {
    QSet<PluginPermission> result;
    for (const auto& s : list) {
        auto p = fromString(s);
        if (p != static_cast<PluginPermission>(0)) {
            result.insert(p);
        }
    }
    return result;
}

}  // namespace PluginPermUtil
```

### 3.3 Step 3：定义插件接口

**新建文件：** `src/core/plugin/IPlugin.h`

```cpp
// src/core/plugin/IPlugin.h
#pragma once

#include <QObject>
#include <QString>

#include "core/plugin/PluginMetadata.h"

class AppContext;

/// 插件抽象接口
///
/// 所有插件必须实现此接口。
/// 宿主通过 QPluginLoader 加载 .dll 后，qobject_cast<IPlugin*> 获取此接口。
///
/// 实现示例：
/// @code
/// class MyPlugin : public QObject, public IPlugin {
///     Q_OBJECT
///     Q_INTERFACES(IPlugin)
///     Q_PLUGIN_METADATA(IID TOOLBOX_PLUGIN_IID FILE "myplugin.json")
/// public:
///     PluginMetadata metadata() const override { ... }
///     bool initialize(AppContext* ctx) override { ... }
///     void shutdown() override { ... }
/// };
/// @endcode
class IPlugin {
public:
    virtual ~IPlugin() = default;

    /// 获取插件元信息（从 JSON 文件中读取）
    [[nodiscard]] virtual PluginMetadata metadata() const = 0;

    /// 初始化插件
    ///
    /// 在插件加载后调用。插件在此方法中：
    /// - 保存 AppContext 指针，获取所需服务
    /// - 注册 EventBus 订阅
    /// - 创建自己的 ViewModel
    ///
    /// @return true 初始化成功，false 初始化失败（插件将被跳过）
    virtual bool initialize(AppContext* ctx) = 0;

    /// 关闭插件
    ///
    /// 在应用退出前调用。插件在此方法中：
    /// - 取消 EventBus 订阅
    /// - 释放资源
    /// - 保存状态
    virtual void shutdown() = 0;

    /// 创建插件的 ViewModel（可选）
    ///
    /// 返回的 QObject* 将被 QML 通过属性访问。
    /// 默认返回 nullptr（无 ViewModel 的插件）。
    [[nodiscard]] virtual QObject* createViewModel() { return nullptr; }

    /// 获取插件主 QML 组件路径（可选）
    ///
    /// 返回 qrc 路径或文件路径，如 "qrc:/plugins/ocr/OcrPage.qml"。
    /// 默认返回空（无 QML 界面的插件）。
    [[nodiscard]] virtual QString qmlMainComponent() { return {}; }
};

/// 插件接口 ID（用于 Q_DECLARE_INTERFACE / Q_PLUGIN_METADATA）
#define IQTOOLS_PLUGIN_IID "com.iqtools.IPlugin/1.0"

Q_DECLARE_INTERFACE(IPlugin, IQTOOLS_PLUGIN_IID)
```

### 3.4 Step 4：实现插件注册表

**新建文件：** `src/core/plugin/PluginRegistry.h`

```cpp
// src/core/plugin/PluginRegistry.h
#pragma once

#include <QObject>
#include <QString>
#include <QReadWriteLock>
#include <memory>
#include <vector>
#include <functional>

#include "core/plugin/IPlugin.h"

/// 已加载插件的运行时信息
struct PluginEntry {
    PluginMetadata                metadata;
    IPlugin*                      instance{nullptr};  // 非拥有指针
    bool                          loaded{false};
    bool                          hasError{false};
    QString                       errorMessage;
};

/// 插件注册表
///
/// 管理所有已加载插件的运行时状态。
/// 线程安全。
class PluginRegistry final : public QObject {
    Q_OBJECT

public:
    explicit PluginRegistry(QObject* parent = nullptr);
    ~PluginRegistry() override;

    /// 注册一个已加载的插件
    void registerPlugin(PluginEntry entry);

    /// 按 ID 查找插件
    [[nodiscard]] PluginEntry* findPlugin(const QString& pluginId);

    /// 获取所有插件列表
    [[nodiscard]] std::vector<PluginEntry> allPlugins() const;

    /// 获取所有已加载且无错误的插件
    [[nodiscard]] std::vector<PluginEntry> loadedPlugins() const;

    /// 检查插件是否已注册
    [[nodiscard]] bool hasPlugin(const QString& pluginId) const;

    /// 启用/禁用插件
    void setPluginEnabled(const QString& pluginId, bool enabled);

    /// 调用所有已加载插件的 shutdown()，然后清理
    void shutdownAll();

    /// 遍历所有已加载插件
    void forEach(const std::function<void(PluginEntry&)>& func);

signals:
    void pluginLoaded(const QString& pluginId, const QString& pluginName);
    void pluginLoadFailed(const QString& pluginId, const QString& reason);
    void pluginUnloaded(const QString& pluginId);

private:
    mutable QReadWriteLock m_lock;
    std::vector<PluginEntry> m_plugins;
};
```

**新建文件：** `src/core/plugin/PluginRegistry.cpp`

```cpp
// src/core/plugin/PluginRegistry.cpp
#include "core/plugin/PluginRegistry.h"

#include "core/log/Logger.h"
#include "core/log/LogModules.h"

PluginRegistry::PluginRegistry(QObject* parent)
    : QObject(parent) {
    TB_LOG_INFO(LogModule::Plugin, "PluginRegistry created");
}

PluginRegistry::~PluginRegistry() {
    shutdownAll();
}

void PluginRegistry::registerPlugin(PluginEntry entry) {
    QWriteLocker locker(&m_lock);

    // 检查重复
    for (const auto& existing : m_plugins) {
        if (existing.metadata.id == entry.metadata.id) {
            TB_LOG_WARN(LogModule::Plugin,
                "Plugin '{}' already registered, skipping",
                entry.metadata.id.toStdString());
            return;
        }
    }

    m_plugins.push_back(std::move(entry));
    const auto& added = m_plugins.back();

    if (added.loaded) {
        TB_LOG_INFO(LogModule::Plugin,
            "Plugin registered: id='{}' name='{}' version={}",
            added.metadata.id.toStdString(),
            added.metadata.name.toStdString(),
            added.metadata.version.toString().toStdString());
        emit pluginLoaded(added.metadata.id, added.metadata.name);
    } else {
        TB_LOG_WARN(LogModule::Plugin,
            "Plugin registered with error: id='{}' error='{}'",
            added.metadata.id.toStdString(),
            added.errorMessage.toStdString());
        emit pluginLoadFailed(added.metadata.id, added.errorMessage);
    }
}

PluginEntry* PluginRegistry::findPlugin(const QString& pluginId) {
    QReadLocker locker(&m_lock);
    for (auto& entry : m_plugins) {
        if (entry.metadata.id == pluginId) {
            return &entry;
        }
    }
    return nullptr;
}

std::vector<PluginEntry> PluginRegistry::allPlugins() const {
    QReadLocker locker(&m_lock);
    return m_plugins;
}

std::vector<PluginEntry> PluginRegistry::loadedPlugins() const {
    QReadLocker locker(&m_lock);
    std::vector<PluginEntry> result;
    for (const auto& e : m_plugins) {
        if (e.loaded && !e.hasError) {
            result.push_back(e);
        }
    }
    return result;
}

bool PluginRegistry::hasPlugin(const QString& pluginId) const {
    QReadLocker locker(&m_lock);
    for (const auto& e : m_plugins) {
        if (e.metadata.id == pluginId) return true;
    }
    return false;
}

void PluginRegistry::setPluginEnabled(const QString& pluginId, bool enabled) {
    QWriteLocker locker(&m_lock);
    for (auto& e : m_plugins) {
        if (e.metadata.id == pluginId) {
            e.metadata.enabled = enabled;
            TB_LOG_INFO(LogModule::Plugin,
                "Plugin '{}' {}", pluginId.toStdString(),
                enabled ? "enabled" : "disabled");
            return;
        }
    }
}

void PluginRegistry::shutdownAll() {
    QWriteLocker locker(&m_lock);
    for (auto& entry : m_plugins) {
        if (entry.loaded && entry.instance) {
            TB_LOG_INFO(LogModule::Plugin,
                "Shutting down plugin '{}'", entry.metadata.id.toStdString());
            entry.instance->shutdown();
            entry.loaded = false;
        }
    }
    m_plugins.clear();
    TB_LOG_INFO(LogModule::Plugin, "All plugins shut down");
}

void PluginRegistry::forEach(const std::function<void(PluginEntry&)>& func) {
    QWriteLocker locker(&m_lock);
    for (auto& entry : m_plugins) {
        if (entry.loaded && !entry.hasError) {
            func(entry);
        }
    }
}
```

### 3.5 Step 5：实现插件加载器

**新建文件：** `src/core/plugin/PluginLoader.h`

```cpp
// src/core/plugin/PluginLoader.h
#pragma once

#include <QObject>
#include <QString>
#include <QDir>
#include <memory>

#include "core/plugin/PluginRegistry.h"

class AppContext;

/// 插件加载器
///
/// 负责：
/// 1. 扫描指定目录下的 .dll/.so/.dylib 文件
/// 2. 使用 QPluginLoader 加载动态库
/// 3. qobject_cast<IPlugin*> 获取插件接口
/// 4. 校验元信息（版本、依赖）
/// 5. 调用 plugin->initialize(appContext)
/// 6. 注册到 PluginRegistry
class PluginLoader final : public QObject {
    Q_OBJECT

public:
    explicit PluginLoader(PluginRegistry* registry,
                          QObject*        parent = nullptr);
    ~PluginLoader() override = default;

    /// 设置插件搜索目录
    void setPluginDir(const QString& dirPath);

    /// 获取当前插件目录
    [[nodiscard]] QString pluginDir() const;

    /// 扫描并加载所有插件
    ///
    /// @param appContext 传给每个插件的 initialize() 方法
    /// @return 成功加载的插件数量
    int loadAll(AppContext* appContext);

    /// 加载单个插件文件
    ///
    /// @param filePath .dll/.so 文件的绝对路径
    /// @param appContext 传给插件的 initialize()
    /// @return true 加载成功
    bool loadPlugin(const QString& filePath, AppContext* appContext);

    /// 卸载指定插件
    bool unloadPlugin(const QString& pluginId);

private:
    /// 校验插件元信息（版本兼容性、依赖满足）
    [[nodiscard]] QString validateMetadata(const PluginMetadata& meta) const;

    /// 检查依赖是否已被加载
    [[nodiscard]] bool checkDependencies(const PluginMetadata& meta) const;

    PluginRegistry* m_registry{nullptr};
    QString         m_pluginDir;
};
```

```cpp
#include <QJsonDocument>

PluginLoader::PluginLoader(PluginRegistry* registry, QObject* parent)
    : QObject(parent)
    , m_registry(registry) {
    TB_LOG_INFO(LogModule::Plugin, "PluginLoader created");
}

void PluginLoader::setPluginDir(const QString& dirPath) {
    m_pluginDir = dirPath;
    TB_LOG_DEBUG(LogModule::Plugin,
        "Plugin directory set to '{}'", dirPath.toStdString());
}

QString PluginLoader::pluginDir() const {
    return m_pluginDir;
}

QString PluginLoader::validateMetadata(const PluginMetadata& meta) const {
    // 检查必填字段
    if (meta.id.isEmpty()) {
        return QStringLiteral("Plugin ID is empty");
    }
    if (meta.name.isEmpty()) {
        return QStringLiteral("Plugin name is empty");
    }
    if (meta.version.isNull()) {
        return QStringLiteral("Plugin version is invalid");
    }

    // 检查最低主程序版本
    if (!meta.minHostVersion.isNull()) {
        const QVersionNumber hostVersion = QVersionNumber::fromString(
            QCoreApplication::applicationVersion());
        if (hostVersion < meta.minHostVersion) {
            return QStringLiteral("Plugin requires host version >= %1, current is %2")
                .arg(meta.minHostVersion.toString(), hostVersion.toString());
        }
    }

    return {};  // 空字符串表示校验通过
}

bool PluginLoader::checkDependencies(const PluginMetadata& meta) const {
    for (const auto& depId : meta.dependencies) {
        if (!m_registry->hasPlugin(depId)) {
            TB_LOG_ERROR(LogModule::Plugin,
                "Plugin '{}' depends on '{}' which is not loaded",
                meta.id.toStdString(), depId.toStdString());
            return false;
        }
    }
    return true;
}

int PluginLoader::loadAll(AppContext* appContext) {
    if (m_pluginDir.isEmpty()) {
        TB_LOG_WARN(LogModule::Plugin, "Plugin directory not set, skipping load");
        return 0;
    }

    QDir dir(m_pluginDir);
    if (!dir.exists()) {
        TB_LOG_WARN(LogModule::Plugin,
            "Plugin directory does not exist: '{}'", m_pluginDir.toStdString());
        return 0;
    }

    // 确定动态库后缀
    QStringList filters;
#if defined(Q_OS_WIN)
    filters << QStringLiteral("*.dll");
#elif defined(Q_OS_MACOS)
    filters << QStringLiteral("*.dylib");
#else
    filters << QStringLiteral("*.so");
#endif

    const QFileInfoList files = dir.entryInfoList(filters, QDir::Files);
    TB_LOG_INFO(LogModule::Plugin,
        "Scanning '{}' | found {} plugin candidates",
        m_pluginDir.toStdString(), files.size());

    int loadedCount = 0;
    for (const QFileInfo& fileInfo : files) {
        if (loadPlugin(fileInfo.absoluteFilePath(), appContext)) {
            ++loadedCount;
        }
    }

    TB_LOG_INFO(LogModule::Plugin,
        "Plugin loading complete | loaded={}/{}", loadedCount, files.size());

    return loadedCount;
}

bool PluginLoader::loadPlugin(const QString& filePath, AppContext* appContext) {
    TB_LOG_DEBUG(LogModule::Plugin,
        "Loading plugin from '{}'", filePath.toStdString());

    QPluginLoader loader(filePath);
    const QJsonObject metaData = loader.metaData();

    // 先尝试不加载实例，只读取元数据
    if (!loader.load()) {
        const QString err = loader.errorString();
        TB_LOG_ERROR(LogModule::Plugin,
            "Failed to load '{}': {}", filePath.toStdString(), err.toStdString());

        PluginEntry entry;
        entry.metadata.id = QFileInfo(filePath).baseName();
        entry.metadata.name = entry.metadata.id;
        entry.loaded = false;
        entry.hasError = true;
        entry.errorMessage = err;
        m_registry->registerPlugin(std::move(entry));
        return false;
    }

    // 获取 IPlugin 接口
    QObject* obj = loader.instance();
    IPlugin* plugin = qobject_cast<IPlugin*>(obj);
    if (!plugin) {
        TB_LOG_ERROR(LogModule::Plugin,
            "'{}' does not implement IPlugin", filePath.toStdString());
        loader.unload();
        return false;
    }

    // 读取元信息
    PluginMetadata meta = plugin->metadata();

    // 如果 JSON 元数据文件中有信息，合并
    if (metaData.contains(QStringLiteral("MetaData"))) {
        const QJsonObject pluginMeta =
            metaData[QStringLiteral("MetaData")].toObject();
        if (meta.id.isEmpty()) {
            meta = PluginMetadata::fromJson(pluginMeta);
        }
    }

    // 校验元信息
    const QString validationErr = validateMetadata(meta);
    if (!validationErr.isEmpty()) {
        TB_LOG_ERROR(LogModule::Plugin,
            "Validation failed for '{}': {}",
            meta.id.toStdString(), validationErr.toStdString());

        PluginEntry entry;
        entry.metadata = meta;
        entry.loaded = false;
        entry.hasError = true;
        entry.errorMessage = validationErr;
        m_registry->registerPlugin(std::move(entry));
        plugin->shutdown();
        loader.unload();
        return false;
    }

    // 检查依赖
    if (!checkDependencies(meta)) {
        PluginEntry entry;
        entry.metadata = meta;
        entry.loaded = false;
        entry.hasError = true;
        entry.errorMessage = QStringLiteral("Missing dependency");
        m_registry->registerPlugin(std::move(entry));
        plugin->shutdown();
        loader.unload();
        return false;
    }

    // 初始化插件
    if (!plugin->initialize(appContext)) {
        TB_LOG_ERROR(LogModule::Plugin,
            "Plugin '{}' initialize() returned false", meta.id.toStdString());

        PluginEntry entry;
        entry.metadata = meta;
        entry.loaded = false;
        entry.hasError = true;
        entry.errorMessage = QStringLiteral("initialize() failed");
        m_registry->registerPlugin(std::move(entry));
        loader.unload();
        return false;
    }

    // 注册成功
    PluginEntry entry;
    entry.metadata = meta;
    entry.instance = plugin;
    entry.loaded   = true;
    entry.hasError = false;
    m_registry->registerPlugin(std::move(entry));

    TB_LOG_INFO(LogModule::Plugin,
        "Plugin loaded successfully: id='{}' name='{}' v{}",
        meta.id.toStdString(), meta.name.toStdString(),
        meta.version.toString().toStdString());

    return true;
}

bool PluginLoader::unloadPlugin(const QString& pluginId) {
    auto* entry = m_registry->findPlugin(pluginId);
    if (!entry || !entry->loaded) {
        return false;
    }

    TB_LOG_INFO(LogModule::Plugin,
        "Unloading plugin '{}'", pluginId.toStdString());

    entry->instance->shutdown();
    entry->loaded = false;

    emit m_registry->pluginUnloaded(pluginId);
    return true;
}
```

```cpp
void OcrViewModel::recognizeFile(const QString& filePath) {
    if (!QFile::exists(filePath)) {
        emit recognizeFailed(QStringLiteral("文件不存在: ") + filePath);
        return;
    }

    setProcessing(true);
    setRecognizedText(QString());

    TB_LOG_INFO(LogModule::Plugin,
        "OCR recognizing | file='{}'", filePath.toStdString());

    // TODO: 接入真实 OCR 引擎（Tesseract / PaddleOCR / 云端 API）

    QTimer::singleShot(1500, this, [this, filePath]() {

            "File: %1\n\n"
            "识别结果示例：\n"
            "这是一段从图片中识别出的文字。\n"
            "The quick brown fox jumps over the lazy dog.\n"
            "1234567890"
        ).arg(filePath);

        setRecognizedText(mockResult);
        emit recognizeSucceeded(mockResult);

        TB_LOG_INFO(LogModule::Plugin, "OCR mock result generated");
    });
}

    if (m_recognizedText.isEmpty()) {

        TB_LOG_DEBUG(LogModule::Plugin, "OCR result copied to clipboard");
    }
}
```

```cpp
// tests/unit/test_plugin_loader.cpp
#include <QtTest/QtTest>
#include <QTemporaryDir>

#include "core/plugin/PluginLoader.h"

    Q_OBJECT

    void shouldHandleMissingDirectory();
    void shouldHandleEmptyDirectory();
    void shouldReportInvalidPlugin();
};

void PluginLoaderTest::shouldHandleMissingDirectory() {
    PluginRegistry registry;
    PluginLoader loader(&registry);

    loader.setPluginDir(QStringLiteral("/nonexistent/path"));
    QCOMPARE(loader.loadAll(nullptr), 0);
}

void PluginLoaderTest::shouldHandleEmptyDirectory() {
    QTemporaryDir dir;
    QVERIFY(dir.isValid());

    PluginRegistry registry;
    PluginLoader loader(&registry);
    loader.setPluginDir(dir.path());

    QCOMPARE(loader.loadAll(nullptr), 0);
    QCOMPARE(registry.allPlugins().size(), 0);
}

void PluginLoaderTest::shouldReportInvalidPlugin() {
    QTemporaryDir dir;
    QVERIFY(dir.isValid());

    // 创建一个假的 .dll 文件（不是真正的动态库）
    QFile fakeFile(dir.path() + "/fake_plugin.dll");
    QVERIFY(fakeFile.open(QIODevice::WriteOnly));
    fakeFile.write("not a real plugin");
    fakeFile.close();

    PluginRegistry registry;
    PluginLoader loader(&registry);
    loader.setPluginDir(dir.path());

    QCOMPARE(loader.loadAll(nullptr), 0);

    // 应该记录了加载失败
    auto plugins = registry.allPlugins();
    QCOMPARE(plugins.size(), 1);
    QVERIFY(plugins[0].hasError);
}

QTEST_MAIN(PluginLoaderTest)
```

### 6.2 插件注册表测试

```cpp
// tests/unit/test_plugin_registry.cpp
#include <QtTest/QtTest>
#include "core/plugin/PluginRegistry.h"

class PluginRegistryTest final : public QObject {
    Q_OBJECT

private slots:
    void shouldRegisterPlugin();
    void shouldRejectDuplicate();
    void shouldFindById();
    void shouldTrackEnabledState();
};

void PluginRegistryTest::shouldRegisterPlugin() {
    PluginRegistry registry;
    QSignalSpy spy(&registry, &PluginRegistry::pluginLoaded);

    PluginEntry entry;
    entry.metadata.id = "test.plugin";
    entry.metadata.name = "Test Plugin";
    entry.loaded = true;

    registry.registerPlugin(entry);
    QCOMPARE(spy.count(), 1);
    QVERIFY(registry.hasPlugin("test.plugin"));
}

void PluginRegistryTest::shouldRejectDuplicate() {
    PluginRegistry registry;

    PluginEntry entry;
    entry.metadata.id = "test.plugin";
    entry.metadata.name = "Test";
    entry.loaded = true;

    registry.registerPlugin(entry);
    registry.registerPlugin(entry);  // 重复注册

    QCOMPARE(registry.allPlugins().size(), 1);
}

void PluginRegistryTest::shouldFindById() {
    PluginRegistry registry;

    PluginEntry entry;
    entry.metadata.id = "com.test.ocr";
    entry.metadata.name = "OCR";
    entry.loaded = true;

    registry.registerPlugin(entry);

    auto* found = registry.findPlugin("com.test.ocr");
    QVERIFY(found != nullptr);
    QCOMPARE(found->metadata.name, QStringLiteral("OCR"));

    QCOMPARE(registry.findPlugin("nonexistent"), nullptr);
}

void PluginRegistryTest::shouldTrackEnabledState() {
    PluginRegistry registry;

    PluginEntry entry;
    entry.metadata.id = "test.plugin";
    entry.loaded = true;
    registry.registerPlugin(entry);

    registry.setPluginEnabled("test.plugin", false);
    auto* found = registry.findPlugin("test.plugin");
    QVERIFY(!found->metadata.enabled);
}

QTEST_MAIN(PluginRegistryTest)
#include "test_plugin_registry.moc"
```

## 7. 常见问题与注意事项

### 7.1 静态链接 vs 动态加载的选择

```
场景 A：内置插件（随主程序一起发布）
  用 CMake STATIC 库，在 main.cpp 中手动实例化。
  不需要 QPluginLoader，不需要 .json 元数据文件。
  适合：官方插件、核心扩展功能。

  // main.cpp
  #include "plugins/plugin_ocr/OcrPlugin.h"

  auto ocrPlugin = std::make_unique<OcrPlugin>();
  ocrPlugin->initialize(&appContext);
  // 注册到 PluginRegistry...

场景 B：外部插件（用户自行安装）
  用 QPluginLoader 动态加载 .dll。
  需要 Q_PLUGIN_METADATA + JSON 文件。
  适合：第三方插件、可选功能。

本指南主要讲解场景 B，因为它是更通用的方案。
```

### 7.2 插件 DLL 的部署位置

```
IQtoolsPlus.exe
├── plugins/
│   ├── plugin_ocr.dll       ← 动态插件
│   ├── plugin_qrcode.dll
│   └── third_party_xxx.dll  ← 用户安装的第三方插件
├── Qt6Core.dll
└── ...
```

在 main.cpp 中设置插件目录时，使用相对于可执行文件的路径：

```cpp
const QString pluginDir = QCoreApplication::applicationDirPath()
                          + QStringLiteral("/plugins");
```

### 7.3 插件与主程序的 ABI 兼容性

```
⚠️ 关键规则：插件和主程序必须使用相同的编译器版本和 Qt 版本。

原因：
- C++ 没有稳定的 ABI（不同编译器版本的类布局可能不同）
- Qt 的 QObject 系统依赖 RTTI 和 vtable，版本不匹配会导致崩溃

建议：
1. 发布插件 SDK 时，明确标注支持的 Qt 版本和编译器版本
2. minHostVersion 字段用于版本兼容性检查
3. 在 CI 中用相同环境编译主程序和官方插件
```

### 7.4 插件中使用 QML_ELEMENT 的注意事项

```cpp
// 插件中的 QML_ELEMENT 需要手动注册，不能依赖自动注册
// 因为插件的 QML 类型在运行时才可用

// 方案 1：在 initialize() 中手动注册
bool OcrPlugin::initialize(AppContext* ctx) {
    qmlRegisterType<OcrViewModel>(
        "IQtools.Plugins.OCR", 1, 0, "OcrViewModel");
    return true;
}

// 方案 2：通过 qmldir 文件声明
// plugins/plugin_ocr/qmldir:
//   module IQtools.Plugins.OCR
//   OcrViewModel 1.0 OcrViewModel.qml
```
