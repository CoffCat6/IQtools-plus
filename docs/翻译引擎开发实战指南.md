## 1. 整体数据流全景图

翻译功能涉及 **4 层**，数据从用户指尖流向翻译 API 再返回：

```
用户操作 ──────────────────────────────────────────────────────────────────────

  ┌──────────────────────────────────────────────────────────────────────┐
  │  TranslatePage.qml                                                  │
  │                                                                      │
  │  用户输入 "Hello World"  →  点击"翻译"按钮                            │
  │                                                                      │
  │  onClicked: root.viewModel.translate()                               │
  │             ▲                                                        │
  │             │ QML 调用 ViewModel 的 Q_INVOKABLE 方法                  │
  └─────────────┼────────────────────────────────────────────────────────┘
                │
QML ↔ C++ 边界 ─┼────────────────────────────────────────────────────────────
                │
  ┌─────────────▼────────────────────────────────────────────────────────┐
  │  TranslateViewModel::translate()                                      │
  │  (src/viewmodels/TranslateViewModel.cpp:132)                         │
  │                                                                      │
  │  1. 构建 TranslateRequest { text, fromLang, toLang, engineId }       │
  │  2. setTranslating(true)  →  emit translatingChanged()               │
  │  3. service->translateAsync(request, callback)                       │
  │                        │                                             │
  │     [等待异步回调...]    │                                             │
  │                        │                                             │
  │  4. 回调触发时：                                                      │
  │     - 成功 → setResultText() + emit translateSucceeded()             │
  │     - 失败 → setErrorMessage() + emit translateFailed()              │
  │     - 最终 → setTranslating(false)                                   │
  └───────────────────────┼──────────────────────────────────────────────┘
                          │ C++ 接口调用
  ┌───────────────────────▼──────────────────────────────────────────────┐
  │  MockTranslateService::translateAsync()                               │
  │  (src/services/translate/MockTranslateService.cpp:15)                │
  │                                                                      │
  │  当前实现：QTimer::singleShot(280ms) → 构造假数据 → callback(ok)      │
  │                                                                      │
  │  未来实现：                                                          │
  │  ① 先查缓存 TranslateCache::get(text, from, to)                     │
  │  ② 缓存命中 → callback(ok(cached_result))                           │
  │  ③ 缓存未命中 → 引擎调用 YoudaoEngine::translateAsync()              │
  │  ④ 引擎返回 → 写缓存 → callback(ok(result))                         │
  └──────────────────────────────────────────────────────────────────────┘

外部 API ──────────────────────────────────────────────────────────────────────

  ┌──────────────────────────────────────────────────────────────────────┐
  │  YoudaoEngine / DeepLEngine  (待实现)                                │
  │                                                                      │
  │  HTTP POST → 翻译 API  →  解析 JSON 响应  →  返回 TranslateResult    │
  └──────────────────────────────────────────────────────────────────────┘
```

## 2. 逐层代码走读

### 2.1 数据结构层 — TranslateTypes.h

**文件：** `src/services/translate/TranslateTypes.h`

这是贯穿整个翻译链路的"数据契约"。所有层都依赖这些结构体，但它们不包含任何逻辑。

```cpp
// 翻译请求：从 ViewModel 传到 Service，再到 Engine
struct TranslateRequest {
    QString text;                    // 待翻译文本
    QString fromLang{"auto"};        // 源语言，"auto" = 自动检测
    QString toLang{"zh-CN"};         // 目标语言
    QString engineId;                // 指定引擎，空 = 使用当前默认
};

// 翻译结果：从 Engine 返回 Service，再返回 ViewModel
struct TranslateResult {
    QString     translatedText;      // 翻译后的文本
    QString     detectedLang;        // 自动检测到的源语言
    QString     engineId;            // 实际使用的引擎 ID
    QStringList alternatives;        // 备选翻译（可选）
    int         latencyMs{0};        // 请求耗时（毫秒）
    QDateTime   timestamp;           // 翻译时间
    bool        cacheHit{false};     // 是否命中缓存
};

// 引擎元信息：用于 UI 展示可用引擎列表
struct TranslateEngineInfo {
    QString id;       // 引擎唯一标识，如 "youdao", "deepl"
    QString name;     // 显示名称，如 "有道翻译"
    bool    available{false};  // 是否可用（API Key 是否配置）
};
```

**设计要点：**
- 使用默认值减少调用方的参数负担
- `cacheHit` 标记让 ViewModel 可以展示缓存命中率
- `engineId` 在结果中返回，让 ViewModel 知道实际用了哪个引擎

### 2.2 接口层 — ITranslateService.h

**文件：** `src/services/translate/ITranslateService.h`

这是 ViewModel 与 Service 之间的"契约"。ViewModel 只认识这个接口，不关心底层实现。

```cpp
class ITranslateService {
public:
    using Ptr = std::shared_ptr<ITranslateService>;  // 智能指针别名
    using TranslateCallback = std::function<void(Result<TranslateResult>)>;

    virtual ~ITranslateService() = default;

    // 核心方法：异步翻译（推荐使用）
    virtual void translateAsync(const TranslateRequest& request,
                                TranslateCallback       callback) = 0;

    // 同步翻译（测试/命令行用）
    [[nodiscard]] virtual Result<TranslateResult>
    translateSync(const TranslateRequest& request) = 0;

    // 引擎管理
    [[nodiscard]] virtual QList<TranslateEngineInfo> availableEngines() const = 0;
    virtual void setCurrentEngine(const QString& engineId) = 0;
    [[nodiscard]] virtual QString currentEngine() const = 0;

    // 辅助
    [[nodiscard]] virtual QStringList supportedLanguages() const = 0;
    [[nodiscard]] virtual double cacheHitRate() const = 0;
};
```

**设计要点：**
- `using Ptr = std::shared_ptr<...>` — 统一智能指针类型，避免裸指针
- `TranslateCallback` 是 `std::function` — 可以传 lambda、函数指针、bind 表达式
- `[[nodiscard]]` — 编译器会警告你忽略了返回值（防止漏检查错误）
- 同时提供异步和同步两种方式 — 异步用于 UI 线程，同步用于测试

### 2.3 错误类型 — Error.h 中的 Result\<T\>

**文件：** `src/core/error/Error.h`

整个项目用 `Result<T>` 替代异常进行错误传播。翻译链路中每一步都可能返回错误：

```cpp
// 使用示例（在 MockTranslateService 中）：
if (trimmed.isEmpty()) {
    // 构造一个失败结果，携带错误码和消息
    callback(Result<TranslateResult>::err(
        TB_MAKE_ERROR(ErrorCode::TranslateEmptyInput,
                      "Text cannot be empty")));
    return;  // 提前返回
}

// 构造成功结果
TranslateResult result;
result.translatedText = "...";
callback(Result<TranslateResult>::ok(std::move(result)));
```

**在 ViewModel 中检查错误：**
```cpp
// TranslateViewModel.cpp:153
svc->translateAsync(request, [this](Result<TranslateResult> result) {
    if (result.isErr()) {
        // 错误路径：显示错误消息
        const QString reason = QString::fromStdString(result.error().message);
        setErrorMessage(reason);
        emit translateFailed(reason);
        return;
    }

    // 成功路径：更新 UI
    const auto& data = result.value();
    setResultText(data.translatedText);
    emit translateSucceeded(m_resultText);
});
```

### 2.4 Mock 实现 — MockTranslateService

**文件：** `src/services/translate/MockTranslateService.h/.cpp`

Mock 的作用：**在没有真实 API 的情况下，让整个 UI 数据流可以跑通**。

```cpp
class MockTranslateService final : public QObject, public ITranslateService {
    Q_OBJECT   // 需要 QObject 因为用了 QTimer
    // ...
};
```

**`translateAsync` 的实现逻辑：**

```
translateAsync(request, callback)
    │
    ├─ 文本为空？
    │   └─ YES → callback(Result::err(TranslateEmptyInput))
    │
    └─ NO → QTimer::singleShot(280ms, ...)
              │
              └─ 构造假的 TranslateResult
                 callback(Result::ok(result))
```

关键点：
- 使用 `QTimer::singleShot` 模拟网络延迟，回调在主线程执行
- 因为回调在主线程，ViewModel 可以直接更新 UI，不需要 `QMetaObject::invokeMethod`
- 真实引擎的回调可能在工作线程，需要处理线程切换

### 2.5 ViewModel 层 — TranslateViewModel

**文件：** `src/viewmodels/TranslateViewModel.h/.cpp`

ViewModel 是 QML 和 Service 之间的桥梁。它做三件事：

**① 暴露状态给 QML（通过 Q_PROPERTY）：**

```cpp
Q_PROPERTY(QString sourceText   READ sourceText   WRITE setSourceText NOTIFY sourceTextChanged FINAL)
Q_PROPERTY(QString resultText   READ resultText   NOTIFY resultTextChanged FINAL)
Q_PROPERTY(bool    translating  READ translating   NOTIFY translatingChanged FINAL)
// ... 还有 fromLanguage, toLanguage, currentEngine, errorMessage 等
```

QML 可以直接读写这些属性，属性变化时自动刷新 UI。

**② 暴露操作给 QML（通过 Q_INVOKABLE）：**

```cpp
Q_INVOKABLE void translate();         // 执行翻译
Q_INVOKABLE void clear();             // 清空输入/输出
Q_INVOKABLE void copyResult();        // 复制结果到剪贴板
Q_INVOKABLE void switchLanguages();   // 交换源/目标语言
```

**③ 管理服务生命周期（injectService / ensureService）：**

```cpp
// 外部注入（AppShellViewModel 调用）
void TranslateViewModel::injectService(ITranslateService::Ptr service) {
    m_service = std::move(service);
    // 同步引擎信息到 UI
    m_currentEngine = m_service->currentEngine();
    emit currentEngineChanged();
}

// 延迟创建（如果没人注入服务）
ITranslateService::Ptr TranslateViewModel::ensureService() {
    if (!m_service) {
        m_service = std::make_shared<MockTranslateService>();
        // ...
    }
    return m_service;
}
```

**translate() 方法详解（核心流程）：**

```cpp
void TranslateViewModel::translate() {
    // 1. 输入校验
    const QString trimmed = m_sourceText.trimmed();
    if (trimmed.isEmpty()) {
        setErrorMessage(tr("请输入待翻译文本。"));
        emit translateFailed(m_errorMessage);
        return;
    }

    // 2. 更新 UI 状态
    setErrorMessage(QString());   // 清除之前的错误
    setTranslating(true);         // 显示加载动画

    // 3. 构建请求
    TranslateRequest request;
    request.text     = trimmed;
    request.fromLang = m_fromLanguage;
    request.toLang   = m_toLanguage;
    request.engineId = m_currentEngine;

    // 4. 调用服务（异步）
    auto svc = ensureService();
    svc->translateAsync(request, [this](Result<TranslateResult> result) {
        // 5. 处理异步回调
        if (result.isErr()) {
            setTranslating(false);
            const QString reason = QString::fromStdString(result.error().message);
            setErrorMessage(reason);
            emit translateFailed(reason);
            return;
        }

        const auto& data = result.value();
        setResultText(data.translatedText);
        setLatencyInfo(tr("%1 ms").arg(data.latencyMs));
        setCacheHitRate(data.cacheHit ? 1.0 : 0.0);
        setTranslating(false);

        emit translateSucceeded(m_resultText);
    });
}
```

**将来替换为真实引擎时，只需修改 main.cpp 中的一行：**

```cpp
// 替换前：
appContext.registerService<ITranslateService>(
    std::make_shared<MockTranslateService>());

// 替换后：
```

ViewModel 和 QML **一行代码都不用改**。这就是接口隔离的威力。

## 3. 如何实现一个真实翻译引擎

### 3.1 目标架构

```
ITranslateService (接口)
```

### 3.2 Step 1：定义引擎接口

**新建文件：** `src/services/translate/engines/ITranslateEngine.h`

```cpp
// src/services/translate/engines/ITranslateEngine.h
#pragma once

#include <QString>
#include <QVariantMap>
#include <functional>
#include <memory>

#include "core/error/Error.h"
#include "services/translate/TranslateTypes.h"

/// 翻译引擎抽象接口
///
/// 每个翻译 API（有道、DeepL、Google 等）实现此接口。
/// 引擎只负责"发送请求 → 解析响应"，不处理缓存和引擎切换。
class ITranslateEngine {
public:
    using Ptr = std::shared_ptr<ITranslateEngine>;
    using TranslateCallback = std::function<void(Result<TranslateResult>)>;

    virtual ~ITranslateEngine() = default;

    /// 引擎唯一标识（如 "youdao", "deepl"）
    [[nodiscard]] virtual QString engineId() const noexcept = 0;

    /// 引擎显示名称（如 "有道翻译", "DeepL"）
    [[nodiscard]] virtual QString engineName() const noexcept = 0;

    /// 引擎是否可用（API Key 是否已配置）
    [[nodiscard]] virtual bool isAvailable() const noexcept = 0;

    /// 异步翻译（回调可能在工作线程中执行）
    virtual void translateAsync(const TranslateRequest& request,
                                TranslateCallback       callback) = 0;

    /// 同步翻译（供测试使用）
    [[nodiscard]] virtual Result<TranslateResult>
    translateSync(const TranslateRequest& request) = 0;

    /// 配置引擎（API Key 等）
    virtual void configure(const QVariantMap& config) = 0;

    /// 获取支持的语言列表
    [[nodiscard]] virtual QStringList supportedLanguages() const = 0;
};
```

### 3.3 Step 2：实现有道翻译引擎

**新建文件：** `src/services/translate/engines/YoudaoEngine.h`

```cpp
// src/services/translate/engines/YoudaoEngine.h
#pragma once

#include <QObject>
#include <QNetworkAccessManager>
#include <QNetworkReply>
#include <memory>

#include "services/translate/engines/ITranslateEngine.h"

/// 有道翻译引擎实现
///
/// 使用有道智云翻译 API v3：
/// https://ai.youdao.com/doc-api.s
///
/// 需要在设置页面配置 appId 和 appSecret。
class YoudaoEngine final : public QObject, public ITranslateEngine {
    Q_OBJECT

public:
    explicit YoudaoEngine(QObject* parent = nullptr);
    ~YoudaoEngine() override = default;

    // ── ITranslateEngine ──────────────────────────────────────────────────
    [[nodiscard]] QString engineId()   const noexcept override;
    [[nodiscard]] QString engineName() const noexcept override;
    [[nodiscard]] bool isAvailable()   const noexcept override;

    void translateAsync(const TranslateRequest& request,
                        TranslateCallback       callback) override;

    [[nodiscard]] Result<TranslateResult>
    translateSync(const TranslateRequest& request) override;

    void configure(const QVariantMap& config) override;
    [[nodiscard]] QStringList supportedLanguages() const override;

private:
    /// 构建 API 请求 URL 和参数
    [[nodiscard]] QUrl buildRequestUrl(const TranslateRequest& request) const;

    /// 构建请求体（表单数据）
    [[nodiscard]] QByteArray buildRequestBody(const TranslateRequest& request) const;

    /// 解析 API 响应
    [[nodiscard]] Result<TranslateResult>
    parseResponse(const QByteArray& responseData, const TranslateRequest& request,
                  int latencyMs) const;

    /// 生成有道 API 签名
    [[nodiscard]] QString generateSign(const QString& input, int salt,
                                       const QString& curtime) const;

    /// 语言代码映射：内部代码 → 有道 API 代码
    [[nodiscard]] QString mapLanguageCode(const QString& langCode) const;

    QNetworkAccessManager m_network;
    QString m_appId;
    QString m_appSecret;
    bool    m_configured{false};
};
```

**新建文件：** `src/services/translate/engines/YoudaoEngine.cpp`

```cpp
// src/services/translate/engines/YoudaoEngine.cpp
#include "services/translate/engines/YoudaoEngine.h"

#include <QJsonDocument>
#include <QJsonObject>
#include <QJsonArray>
#include <QUrlQuery>
#include <QMessageAuthenticationCode>
#include <QElapsedTimer>
#include <QRandomGenerator>

#include "core/log/Logger.h"
#include "core/log/LogModules.h"

YoudaoEngine::YoudaoEngine(QObject* parent)
    : QObject(parent) {
    TB_LOG_INFO(LogModule::Translate, "YoudaoEngine created");
}

QString YoudaoEngine::engineId() const noexcept {
    return QStringLiteral("youdao");
}

QString YoudaoEngine::engineName() const noexcept {
    return QStringLiteral("有道翻译");
}

bool YoudaoEngine::isAvailable() const noexcept {
    return m_configured && !m_appId.isEmpty() && !m_appSecret.isEmpty();
}

void YoudaoEngine::configure(const QVariantMap& config) {
    m_appId     = config.value(QStringLiteral("appId")).toString();
    m_appSecret = config.value(QStringLiteral("appSecret")).toString();
    m_configured = true;

    TB_LOG_INFO(LogModule::Translate,
        "YoudaoEngine configured | appId={} available={}",
        m_appId.left(6).toStdString(), isAvailable());
}

QStringList YoudaoEngine::supportedLanguages() const {
    return {
        QStringLiteral("auto"),
        QStringLiteral("zh-CHS"),   // 中文
        QStringLiteral("en"),       // 英语
        QStringLiteral("ja"),       // 日语
        QStringLiteral("ko"),       // 韩语
        QStringLiteral("fr"),       // 法语
        QStringLiteral("de"),       // 德语
        QStringLiteral("ru"),       // 俄语
        QStringLiteral("es"),       // 西班牙语
        QStringLiteral("pt"),       // 葡萄牙语
    };
}

QString YoudaoEngine::mapLanguageCode(const QString& langCode) const {
    // 内部统一代码 → 有道 API 代码
    static const QMap<QString, QString> mapping = {
        {QStringLiteral("zh-CN"),  QStringLiteral("zh-CHS")},
        {QStringLiteral("zh-TW"),  QStringLiteral("zh-CHT")},
        {QStringLiteral("auto"),   QStringLiteral("auto")},
    };
    return mapping.value(langCode, langCode);
}

QString YoudaoEngine::generateSign(const QString& input, int salt,
                                    const QString& curtime) const {
    // 有道 API v3 签名算法：
    // sign = sha256(appId + input + salt + curtime + appSecret)
    const QString truncated = input.length() <= 20
        ? input
        : input.left(10) + QString::number(input.length()) + input.right(10);

    const QString signStr = m_appId + truncated + QString::number(salt)
                            + curtime + m_appSecret;

    return QString::fromLatin1(
        QCryptographicHash::hash(signStr.toUtf8(),
                                 QCryptographicHash::Sha256).toHex());
}

QUrl YoudaoEngine::buildRequestUrl(const TranslateRequest& /*request*/) const {
    return QUrl(QStringLiteral("https://openapi.youdao.com/api"));
}

QByteArray YoudaoEngine::buildRequestBody(const TranslateRequest& request) const {
    const qint64 curtimeSec = QDateTime::currentSecsSinceEpoch();
    const int salt = QRandomGenerator::global()->bounded(10000, 99999);
    const QString curtime = QString::number(curtimeSec);
    const QString sign = generateSign(request.text, salt, curtime);

    QUrlQuery params;
    params.addQueryItem(QStringLiteral("q"),      request.text);
    params.addQueryItem(QStringLiteral("from"),   mapLanguageCode(request.fromLang));
    params.addQueryItem(QStringLiteral("to"),     mapLanguageCode(request.toLang));
    params.addQueryItem(QStringLiteral("appKey"), m_appId);
    params.addQueryItem(QStringLiteral("salt"),   QString::number(salt));
    params.addQueryItem(QStringLiteral("sign"),   sign);
    params.addQueryItem(QStringLiteral("signType"), QStringLiteral("v3"));
    params.addQueryItem(QStringLiteral("curtime"), curtime);

    return params.toString(QUrl::FullyEncoded).toUtf8();
}

Result<TranslateResult>
YoudaoEngine::parseResponse(const QByteArray& responseData,
                             const TranslateRequest& request,
                             int latencyMs) const {
    QJsonParseError parseErr;
    const auto doc = QJsonDocument::fromJson(responseData, &parseErr);
    if (parseErr.error != QJsonParseError::NoError) {
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateApiError,
                          "Invalid JSON response from Youdao"));
    }

    const auto root = doc.object();

    // 检查错误码
    const QString errorCode = root[QStringLiteral("errorCode")].toString();
    if (errorCode != QStringLiteral("0")) {
        TB_LOG_ERROR(LogModule::Translate,
            "Youdao API error: errorCode={}", errorCode.toStdString());
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateApiError,
                          "Youdao API error: " + errorCode.toStdString()));
    }

    // 解析翻译结果
    const QJsonArray transResults = root[QStringLiteral("translation")].toArray();
    if (transResults.isEmpty()) {
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateApiError,
                          "Empty translation result from Youdao"));
    }

    TranslateResult result;
    result.translatedText = transResults[0].toString();
    result.detectedLang   = root[QStringLiteral("lSrc")].toString();
    result.engineId       = engineId();
    result.latencyMs      = latencyMs;
    result.timestamp      = QDateTime::currentDateTime();
    result.cacheHit       = false;

    // 解析备选翻译（如果有）
    if (root.contains(QStringLiteral("web"))) {
        const QJsonArray webResults = root[QStringLiteral("web")].toArray();
        for (const auto& web : webResults) {
            const QJsonArray values = web.toObject()
                [QStringLiteral("value")].toArray();
            for (const auto& v : values) {
                result.alternatives.append(v.toString());
            }
        }
    }

    return Result<TranslateResult>::ok(std::move(result));
}

void YoudaoEngine::translateAsync(const TranslateRequest& request,
                                   TranslateCallback       callback) {
    if (!isAvailable()) {
        callback(Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateAuthError,
                          "Youdao engine not configured. Please set appId and appSecret.")));
        return;
    }

    if (request.text.trimmed().isEmpty()) {
        callback(Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateEmptyInput,
                          "Text cannot be empty")));
        return;
    }

    QElapsedTimer timer;
    timer.start();

    QNetworkRequest netRequest(buildRequestUrl(request));
    netRequest.setHeader(QNetworkRequest::ContentTypeHeader,
                         QStringLiteral("application/x-www-form-urlencoded"));

    QNetworkReply* reply = m_network.post(netRequest,
                                          buildRequestBody(request));

    // 注意：回调在 Qt 网络线程中执行
    QObject::connect(reply, &QNetworkReply::finished, this,
        [this, reply, request, callback, timer]() {
            reply->deleteLater();

            const int latencyMs = static_cast<int>(timer.elapsed());

            if (reply->error() != QNetworkReply::NoError) {
                TB_LOG_ERROR(LogModule::Translate,
                    "Youdao HTTP error: {} {}",
                    static_cast<int>(reply->error()),
                    reply->errorString().toStdString());
                callback(Result<TranslateResult>::err(
                    TB_MAKE_ERROR(ErrorCode::NetworkError,
                                  reply->errorString().toStdString())));
                return;
            }

            auto result = parseResponse(reply->readAll(), request, latencyMs);

            if (result.isOk()) {
                TB_LOG_INFO(LogModule::Translate,
                    "Youdao translation OK | latency={}ms from={} to={}",
                    latencyMs, request.fromLang.toStdString(),
                    request.toLang.toStdString());
            }

            callback(std::move(result));
        });
}

Result<TranslateResult>
YoudaoEngine::translateSync(const TranslateRequest& request) {
    if (!isAvailable()) {
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateAuthError,
                          "Youdao engine not configured"));
    }

    QElapsedTimer timer;
    timer.start();

    QNetworkRequest netRequest(buildRequestUrl(request));
    netRequest.setHeader(QNetworkRequest::ContentTypeHeader,
                         QStringLiteral("application/x-www-form-urlencoded"));

    // 同步请求（阻塞当前线程，仅用于测试/CLI）
    QNetworkReply* reply = m_network.post(netRequest,
                                          buildRequestBody(request));

    // 使用事件循环等待完成
    QEventLoop loop;
    QObject::connect(reply, &QNetworkReply::finished, &loop, &QEventLoop::quit);
    loop.exec();

    const int latencyMs = static_cast<int>(timer.elapsed());
    reply->deleteLater();

    if (reply->error() != QNetworkReply::NoError) {
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::NetworkError,
                          reply->errorString().toStdString()));
    }

    return parseResponse(reply->readAll(), request, latencyMs);
}
```

```cpp
#include <QObject>

/// 真实翻译服务
///
/// 职责：
/// 1. 管理多个翻译引擎
/// 2. 按用户选择调度引擎
/// 3. LRU 缓存翻译结果
/// 4. 引擎失败时自动降级到下一个可用引擎
class TranslateService final : public QObject, public ITranslateService {
    Q_OBJECT

public:
    explicit TranslateService(QObject* parent = nullptr);
    ~TranslateService() override = default;

    /// 注册翻译引擎
    void registerEngine(ITranslateEngine::Ptr engine);

    // ── ITranslateService ──────────────────────────────────────────────────

    void setCurrentEngine(const QString& engineId) override;
    [[nodiscard]] QString currentEngine() const override;
    [[nodiscard]] QStringList supportedLanguages() const override;
    [[nodiscard]] double cacheHitRate() const override;

private:
    /// 生成缓存键
    [[nodiscard]] static QString cacheKey(const TranslateRequest& request);

    /// 查找指定引擎
    [[nodiscard]] ITranslateEngine::Ptr findEngine(const QString& engineId) const;

    std::vector<ITranslateEngine::Ptr> m_engines;
    QString m_currentEngineId;
    mutable QReadWriteLock m_lock;

    // 缓存统计
    int64_t m_totalRequests{0};
    int64_t m_cacheHits{0};
};
```

```cpp
// src/services/translate/TranslateService.cpp
#include "services/translate/TranslateService.h"

#include "core/log/Logger.h"
#include "core/log/LogModules.h"

#include <QElapsedTimer>

TranslateService::TranslateService(QObject* parent)
    : QObject(parent) {
    TB_LOG_INFO(LogModule::Translate, "TranslateService created");
}

void TranslateService::registerEngine(ITranslateEngine::Ptr engine) {
    if (!engine) return;

    QWriteLocker locker(&m_lock);
    m_engines.push_back(std::move(engine));

    // 第一个注册的引擎设为默认
    if (m_engines.size() == 1) {
        m_currentEngineId = m_engines.front()->engineId();
    }

    TB_LOG_INFO(LogModule::Translate,
        "Engine registered: id='{}' name='{}' available={}",
        m_engines.back()->engineId().toStdString(),
        m_engines.back()->engineName().toStdString(),
        m_engines.back()->isAvailable());
}

ITranslateEngine::Ptr
TranslateService::findEngine(const QString& engineId) const {
    for (const auto& engine : m_engines) {
        if (engine->engineId() == engineId) {
            return engine;
        }
    }
    return nullptr;
}

QString TranslateService::cacheKey(const TranslateRequest& request) {
    return QStringLiteral("%1|%2|%3")
        .arg(request.fromLang, request.toLang, request.text);
}

void TranslateService::translateAsync(const TranslateRequest& request,
                                       TranslateCallback       callback) {
    const QString trimmed = request.text.trimmed();
    if (trimmed.isEmpty()) {
        callback(Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateEmptyInput,
                          "Text cannot be empty")));
        return;
    }

    {
        QReadLocker locker(&m_lock);
        m_totalRequests++;
    }

    // 选择引擎：优先使用请求指定的引擎，否则使用当前默认引擎
    const QString targetEngineId = request.engineId.isEmpty()
        ? m_currentEngineId
        : request.engineId;

    auto engine = findEngine(targetEngineId);
    if (!engine) {
        // 引擎不存在，降级到第一个可用引擎
        for (const auto& e : m_engines) {
            if (e->isAvailable()) {
                engine = e;
                break;
            }
        }
    }

    if (!engine) {
        callback(Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateApiError,
                          "No available translation engine")));
        return;
    }

    TB_LOG_DEBUG(LogModule::Translate,
        "Dispatching to engine '{}' | text='{}' from={} to={}",
        engine->engineId().toStdString(),
        trimmed.left(50).toStdString(),
        request.fromLang.toStdString(),
        request.toLang.toStdString());

    // 调用引擎（回调可能在工作线程中）
    engine->translateAsync(request, [this, callback,
                                      targetEngineId](Result<TranslateResult> result) {
        if (result.isOk()) {
            auto data = std::move(result).value();
            data.engineId = targetEngineId;  // 确保引擎 ID 正确
            callback(Result<TranslateResult>::ok(std::move(data)));
        } else {
            callback(std::move(result));
        }
    });
}

Result<TranslateResult>
TranslateService::translateSync(const TranslateRequest& request) {
    auto engine = findEngine(m_currentEngineId);
    if (!engine) {
        return Result<TranslateResult>::err(
            TB_MAKE_ERROR(ErrorCode::TranslateApiError,
                          "No available translation engine"));
    }
    return engine->translateSync(request);
}

QList<TranslateEngineInfo> TranslateService::availableEngines() const {
    QReadLocker locker(&m_lock);
    QList<TranslateEngineInfo> list;
    for (const auto& engine : m_engines) {
        list.append({
            .id        = engine->engineId(),
            .name      = engine->engineName(),
            .available = engine->isAvailable(),
        });
    }
    return list;
}

void TranslateService::setCurrentEngine(const QString& engineId) {
    QWriteLocker locker(&m_lock);
    if (m_currentEngineId != engineId) {
        m_currentEngineId = engineId;
        TB_LOG_INFO(LogModule::Translate,
            "Default engine switched to '{}'", engineId.toStdString());
    }
}

QString TranslateService::currentEngine() const {
    QReadLocker locker(&m_lock);
    return m_currentEngineId;
}

QStringList TranslateService::supportedLanguages() const {
    // 合并所有引擎支持的语言
    QReadLocker locker(&m_lock);
    QSet<QString> langs;
    for (const auto& engine : m_engines) {
        for (const auto& lang : engine->supportedLanguages()) {
            langs.insert(lang);
        }
    }
    return QStringList(langs.begin(), langs.end());
}

double TranslateService::cacheHitRate() const {
    QReadLocker locker(&m_lock);
    if (m_totalRequests == 0) return 0.0;
    return static_cast<double>(m_cacheHits) / m_totalRequests;
}
```

## 5. 常见问题与注意事项

### 5.1 线程安全

```
问题：YoudaoEngine 的网络回调在哪个线程？

答案：QNetworkAccessManager 使用 Qt 内部线程池。
      但因为我们用了 QObject::connect + lambda，
      且 YoudaoEngine 继承 QObject，
      回调会在 YoudaoEngine 对象所在的线程执行。

      如果 YoudaoEngine 在主线程创建（main.cpp 中），
      回调也在主线程 → 可以安全调用 ViewModel。

      如果在工作线程创建 → 需要用 QMetaObject::invokeMethod 切回主线程。
```

**当前项目的约定：** Service 对象在主线程创建，网络回调也在主线程。ViewModel 可以直接在回调中更新状态。

### 5.2 回调中捕获 this 指针

```cpp
// TranslateViewModel.cpp:153
svc->translateAsync(request, [this](Result<TranslateResult> result) {
    // 这里的 this 是 TranslateViewModel*
    // 如果 ViewModel 在异步回调触发前被销毁 → 悬垂指针 → 崩溃！
});
```

**当前风险：** Mock 使用 QTimer::singleShot(280ms)，在正常操作中 ViewModel 不会在 280ms 内被销毁，所以暂时安全。

**更安全的做法（未来改进）：** 使用 `QPointer` 或 `weak_ptr`：

```cpp
auto self = QPointer<TranslateViewModel>(this);
svc->translateAsync(request, [self](Result<TranslateResult> result) {
    if (!self) return;  // ViewModel 已销毁，静默退出
    // ... 正常处理
});
```
